  <p>The ability to open up files such as plain text or CSV files is a great asset 
    for you as a programmer. Not every job requires a database with multiple tables, 
    and storing simple data in a text file can be a good alternative. Especially 
    if your web host doesn't allow you to have a database!</p>
  <p>In this section, we take a look at the various way you can manipulate files 
    with PHP. We'll start by opening a plain text file.</p>
  <p>For this section, we'll be using the file called <b>dictionary.txt</b>, which 
    is amongst <a href="../downloads.html">the ones you downloaded</a> (in the 
    <b>scripts</b> folder).</p>
  <p>Open this file up, and take a look at the contents. It should look like this:</p>
<P class="indent_big_no_bold"><b>AAS</b> = Alive and smiling <br>
    <b>ADN</b> = Any day now <br>
    <b>AEAP</b> = As early as possible <br>
    <b>AFAIK</b> = As far as I know <br>
    <b>AFK</b> = Away from keyboard <br>
    <b>AKA</b> = Also known as</P>
  <p>It's just a long list of values separated by an equals sign. On the left 
    of the equals sign is an abbreviation, and on the right its meaning. We'll 
    open this text file with PHP code. You'll see different ways to open up and 
    work with this text file. Click below to make a start.</p>
