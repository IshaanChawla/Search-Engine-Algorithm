<h1>CSS Introduction</h1>
<p>It is time to take your web designing skills to the next level with <b>C</b>ascading <b>S</b>tyle <b>S</b>heets (CSS). They are a way to control the look and feel of your HTML documents in an organized and efficient manner.  With CSS you will be able to:
<ul>
	<li>Add new looks to your old HTML</li>
	<li>Completely restyle a web site with only a few changes to your CSS code</li>
	<li>Use the "style" you create on any webpage you wish!</li>
</ul>
</p>

</p>

<h1>Intended Audience</h1>
<p>Before you begin the CSS Tutorial we suggest that you
check to see you meet the following recommendations:</p>
<ul>
	<li>You have used HTML in the past</li>
	<li>You know the basic HTML tags and vocabulary.</li>
	<li>You want to be a better web designer!</li>
</ul>

<p>If you said no to one of the above, we recommend that you check out our <a HREF="http://www.tizag.com/htmlT/">HTML Tutorial</a> before taking on CSS.</p>
