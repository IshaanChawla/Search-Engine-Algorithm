      <hr />

      <div class="column" id="cloud-sql">
        <h4>Cloud SQL</h4>
        <p>
          Store and manage data using a relational MySQL database. Google
          handles replication, patch management and database management to
          ensure availability and performance, and you can even have your
          instance automatically co-locate with your deployed applications.
        </p>
        <ol class="toc">
          <li><a href="/sql/docs/introduction">Overview</a></li>

      <div class="column" id="cloud-storage">
        <h4>Cloud Storage</h4>
        <p>
          Use a durable and highly available object storage service. With global
          edge-caching, your users have fast access to your app’s data from any
          location. Google manages versioning, guarantees a strong SLA and
          provides a simple API that allows you to manage your data
          programmatically.
        </p>
        <ol class="toc">
          <li><a href="/storage/docs/overview">Overview</a></li>

        <h4>Cloud Datastore</h4>
        <p>
          Cloud Datastore provides a managed, NoSQL, schemaless database for
          storing non-relational data. Cloud Datastore automatically scales as
          you need it and supports transactions as well as robust, SQL-like
          queries.
        </p>
        <ol class="toc">
          <li><a href="/datastore/docs/concepts/overview">Overview</a></li>
          <li><a href="/datastore/docs/getstarted/overview">Tutorial</a></li>
          <li><a href="/datastore/docs">Documentation</a></li>
         </ol>
      </div>

      <div class="column" id="cloud-bigtable">
        <h4>Cloud Bigtable</h4>
        <p>
          Cloud Bigtable offers a fast, fully managed, massively scalable NoSQL
          database service that's ideal for web, mobile, and Internet of Things
          applications that require terabytes to petabytes of data. Cloud
          Bigtable's underlying technology has been battle-tested at Google for
          more than 10 years—it's the storage system that drives major
          applications such as Google Analytics and Gmail.
        </p>
        <ol class="toc">
          <li><a href="/bigtable/docs/api-overview">Overview</a></li>

    <section id="services">
      <h3>Services</h3>
      <hr />

      <div class="column three" id="cloud-endpoints">
        <h4>Cloud Endpoints</h4>
        <p>
          Create RESTful services from your code and make them accessible to
          iOS, Android and Javascript clients. Automatically generate client
          libraries to make wiring up the frontend easy.
        </p>
        <ol class="toc">
          <li><a href="/appengine/docs/python/endpoints/">Overview</a></li>
          <li>
            Tutorials:
            <a href="/appengine/docs/python/endpoints/getstarted/">Python</a>,
            <a href="/appengine/docs/java/endpoints/getstarted/">Java</a>
          </li>
          <li><a href="/appengine/features/#Endpoints">Documentation</a></li>
        </ol>
      </div>

      <div class="column three" id="cloud-pubsub">
        <h4>Cloud Pub/Sub</h4>
        <p>
          Google Cloud Pub/Sub is designed to provide reliable, many-to-many,
          asynchronous messaging between applications. Publisher applications
          can send messages to a "topic" and other applications can subscribe to
          that topic to receive the messages.
        </p>
        <ol class="toc">
          <li><a href="/pubsub/overview">Overview</a></li>
          <li><a href="/pubsub/getting-started">Tutorial</a></li>
          <li><a href="/pubsub/docs">Documentation</a></li>
        </ol>
      </div>

      <div class="column three" id="cloud-dns">
        <h4>Cloud DNS</h4>
        <p>
          A highly available and scalable DNS service designed to give
          developers and businesses an extremely reliable and cost effective way
          to route end users to Internet applications and services.
        </p>
        <ol class="toc">
          <li><a href="/dns/what-is-cloud-dns">Overview</a></li>
          <li><a href="/dns/getting-started">Tutorial</a></li>
          <li><a href="/dns/docs">Documentation</a></li>
        </ol>
      </div>

      <div class="column three" id="cloud-monitoring">
        <h4>Cloud Monitoring</h4>
        <p>
          The Google Cloud Monitoring API allows you to read monitoring data
          such as response times, uptime, disk usage, MySQL queries, and more
          for your Google Cloud Platform projects.
        </p>
        <ol class="toc">
          <li><a href="/monitoring/what-is-cloud-monitoring">Overview</a></li>
          <li><a href="/monitoring/getting-started">Tutorial</a></li>
          <li><a href="/monitoring/docs">Documentation</a></li>
        </ol>
      </div>

      <div class="column three" id="prediction-api">
        <h4>Prediction API</h4>
        <p>
          Use Google's machine learning algorithms to analyze data and predict
          future outcomes using a familiar RESTful interface. Make product
          recommendations to users, detect spam, analyze text for sentiment, and
          more.
        </p>
        <ol class="toc">
          <li><a href="/prediction/docs/getting-started">Overview</a></li>
          <li><a href="/prediction/docs/hello_world">Tutorial</a></li>
          <li><a href="/prediction/docs">Documentation</a></li>
        </ol>
      </div>

      <div class="column three" id="translate-api">
        <h4>Translate API</h4>
        <p>
          Create multilingual apps and translate text into other languages
          programmatically. Thousands of language pairs are available.
        </p>
        <ol class="toc">
          <li><a href="/translate/">Overview</a></li>
          <li><a href="/translate/v2/getting_started">Tutorial</a></li>
          <li><a href="/translate/docs">Documentation</a></li>
        </ol>
      </div>

    </section>

          Cloud Platform. Use your Cloud Repository as a remote to a local Git
          repo, or sync it with a repo hosted on GitHub.
        </p>

      </div>

      <div class="column three" id="source-code-tools">
