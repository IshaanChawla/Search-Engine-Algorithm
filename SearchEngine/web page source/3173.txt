<p>Welcome to Tizag.com's introduction to HTML and web design.  This short tutorial is aimed to give newbies a little experience in
writing HTML code, saving their files correctly, and viewing the completed works in a web browser.  Regrettably this tutorial cannot
teach you the basics of using a computer, so please be sure that you meet the following requirements:</p>

<h1>Tutorial Overview</h1>
<p>In this tutorial you will be transcribing code into notepad and then viewing it with a web browser.
The code is called HTML (Hyper Text Markup Language) and notepad is a commonly used text editor on Window PCs.  HTML
may seem confusing at first, but we will help you understand how it works in this step-by-step tutorial of how to make
your first web page.</p>

<h1>Your First Web Page</h1>
<p>
To start off with copy the following HTML code into notepad.  Be sure to copy the code exactly, otherwise your
web page may not function correctly.
</p>

<div class='codewrap'><h2 class='specialT'>HTML Code:</h2><pre class='code'>
&lt;html&gt;

&lt;head&gt;
&lt;/head&gt;

&lt;body&gt;

&lt;h2&gt;My first webpage!&lt;/h2&gt;

&lt;/body&gt;
&lt;/html&gt;

</pre></div>

<p>The above code is all that is required to create a basic web page!  Now save your file in notepad by selecting <b>Menu</b> and then <b>Save</b>.  
Click on the <b>Save as Type</b> drop down box and select the option
<b>All Files</b>.  
</p>
<p>When asked to name your file, type "index.html", without the quotes.  Double check that you did everything correctly
and then press save.  Remember where it was saved to because you will need to open this file, soon!
</p>

<h1>Viewing Your Web Page - Web Browsers</h1>
<p>To view your web page, you are going to have to use a web browser (of course).
Web browsers are programs that interpret HTML, like what you have just copied into notepad, 
and transform that code into a visual representation, or a web page.  Common web 
browsers include:</p>
<ul>
<li> <a HREF="http://www.microsoft.com/windows/ie/" TARGET="_blank"><b>Internet Explorer</b></a></li> 
<li><a HREF="http://www.mozilla.org/products/firefox/" TARGET="_blank"><b>FireFox</b></a></li>
<li><a HREF="http://www.opera.com/" TARGET="_blank"><b>Opera</b></a></li>
<li>Netscape Navigator - R.I.P.</li>
</ul>
<h1>Viewing your page</h1>

<p>To view your web page, you must open the "index.html" file inside of a web browser.
Open up another browser window and then follow these instruction.  </p>
<ol>
<li>In the new browser window, select <b>File</b> then <b>Open</b></li>
<li>Then click <b>Browse</b> to enter Windows Explorer</li>
<li>Do you remember where you file is?  Good, then navigate to its location</li>
<li>When you find your file, index.html, double-click the file
to open it inside your web browser</li>
</ol>
<p>Success!  You have just viewed your very first webpage.  <br /><br />If this did not work for you, 
please go through the steps again and follow the directions closely.
If you still can't get this to work, please <a HREF="http://www.tizag.com/about/contact.php">Contact Us</a> and we will get you up and running.
</p>
