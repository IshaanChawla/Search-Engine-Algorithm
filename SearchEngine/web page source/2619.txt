                  <div>
                     <h2 class="title"><a name="overview"></a>1.&nbsp;Spring JDBC
                     </h2>
                  </div>
               </div>
            </div>
            <p> Spring provides a simplification in handling database access
               with the Spring JDBC Template.
            </p>
            <p> The Spring JDBC Template has the following advantages compared
               with standard JDBC.
            </p>
            <p>
               
            </p>
            <div class="itemizedlist">
               <ul class="itemizedlist" type="disc">
                  <li class="listitem">
                     <p>The Spring JDBC template allows to clean-up the resources
                        automatically, e.g. release the database connections.
                     </p>
                  </li>
                  <li class="listitem">
                     <p>The Spring JDBC template converts the standard JDBC
                        SQLExceptions into RuntimeExceptions. This allows the programmer to
                        react more flexible to the errors. The Spring JDBC template
                        converts also the vendor specific error messages into better
                        understandable error messages.
                     </p>
                  </li>
               </ul>
            </div>
            <p>
               
            </p>
            <p> The Spring JDBC template offers several ways to query the
               database. queryForList() returns a list of HashMaps. The name of the
               column is the key in the hashmap for the values in the table.
            </p>
            <p> More convenient is the usage of ResultSetExtractor or RowMapper
               which allows to translates the SQL result direct into an object
               (ResultSetExtractor) or a list of objects (RowMapper). Both these
               methods will be demonstrated in the coding.
            </p>
         </div>

            </p>
            <p> To use the Spring JDBC template you need to add spring-jdbc.jar
               to your classpath (in addition to the standard jars described in the
               above installation link).
            </p>

                     <h2 class="title"><a name="jdbc_usage"></a>3.&nbsp;Using Spring JDBC template
                     </h2>
                  </div>
               </div>
            </div>
            <p> The following example will demonstrate the usage of the JDBC
               template. The example is based on derby.
            </p>
            <div class="section">
               <div class="titlepage">
                  <div>
                     <div>
                        <h3 class="title"><a name="jdbc_usage_db"></a>3.1.&nbsp;Create Database
                        </h3>
                     </div>
                  </div>
               </div>
               <p>
                  Create a small derby database under c:\temp\databases\test01 with the
                  table "Person". Please see
                  <a class="ulink" href="http://www.vogella.com/tutorials/EclipseDataToolsPlatform/article.html" target="_top"> Handling Database with the Eclipse Data Tools Project</a>
                  for an explanation on how to do this with the Eclipse Data Tools
                  Project.
                  
               </p>
               <p> Create the table Person with the following SQL statement.
                  
               </p>
               <p>
                  
               </p><pre class="programlisting">CREATE TABLE Person(ID <span class="hl-keyword">int</span> generated by <span class="hl-keyword">default</span> as identity (start with <span class="hl-number">1</span>) not null,
    FIRSTNAME VARCHAR(<span class="hl-number">20</span>) NOT NULL,
    LASTNAME VARCHAR(<span class="hl-number">20</span>) NOT NULL,
    PRIMARY KEY (ID)); </pre><p>
                  
               </p>

<span class="hl-keyword">import</span> java.util.List;

<span class="hl-keyword">import</span> javax.sql.DataSource;

<span class="hl-keyword">import</span> domainmodel.Person;

<span class="hl-keyword">public</span> <span class="hl-keyword">interface</span> IDao {

  List&lt;Person&gt; select(String firstname, String lastname);

  List&lt;Person&gt; selectAll();

                     <div>
                        <h3 class="title"><a name="jdbc_usage_dao"></a>3.4.&nbsp;Create the Dao with the JDBC Template
                        </h3>
                     </div>
                  </div>
               </div>
               <p> Now you can create the Dao using the JDBC Template function.
                  
               </p>
               <p>
                  
               </p><pre class="programlisting"><span class="hl-keyword">package</span> dao.mapper;

<span class="hl-keyword">import</span> java.sql.ResultSet;
<span class="hl-keyword">import</span> java.sql.SQLException;

<span class="hl-keyword">import</span> org.springframework.jdbc.core.ResultSetExtractor;

<span class="hl-keyword">import</span> domainmodel.Person;

<span class="hl-keyword">public</span> <span class="hl-keyword">class</span> PersonResultSetExtractor <span class="hl-keyword">implements</span> ResultSetExtractor {

  <em><span class="hl-annotation" style="color: gray">@Override</span></em>
  <span class="hl-keyword">public</span> Object extractData(ResultSet rs) <span class="hl-keyword">throws</span> SQLException {
    Person person = <span class="hl-keyword">new</span> Person();
    person.setFirstName(rs.getString(<span class="hl-number">1</span>));
    person.setLastName(rs.getString(<span class="hl-number">2</span>));
    <span class="hl-keyword">return</span> person;
  }

} </pre><p>
                  
               </p>
               <p>
                  
               </p><pre class="programlisting"><span class="hl-keyword">package</span> dao.mapper;

<span class="hl-keyword">import</span> java.sql.ResultSet;
<span class="hl-keyword">import</span> java.sql.SQLException;

<span class="hl-keyword">import</span> org.springframework.jdbc.core.RowMapper;

<span class="hl-keyword">public</span> <span class="hl-keyword">class</span> PersonRowMapper <span class="hl-keyword">implements</span> RowMapper {

  <em><span class="hl-annotation" style="color: gray">@Override</span></em>
  <span class="hl-keyword">public</span> Object mapRow(ResultSet rs, <span class="hl-keyword">int</span> line) <span class="hl-keyword">throws</span> SQLException {
    PersonResultSetExtractor extractor = <span class="hl-keyword">new</span> PersonResultSetExtractor();
    <span class="hl-keyword">return</span> extractor.extractData(rs);
  }

} </pre><p>
                  
               </p>
               <p>
                  
               </p><pre class="programlisting"><span class="hl-keyword">package</span> dao;

<span class="hl-keyword">import</span> java.util.List;

<span class="hl-keyword">import</span> javax.sql.DataSource;

<span class="hl-keyword">import</span> org.springframework.jdbc.core.JdbcTemplate;

<span class="hl-keyword">import</span> dao.mapper.PersonRowMapper;
<span class="hl-keyword">import</span> domainmodel.Person;

<span class="hl-keyword">public</span> <span class="hl-keyword">class</span> DerbyDao <span class="hl-keyword">implements</span> IDao {
  <span class="hl-keyword">private</span> DataSource dataSource;

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> setDataSource(DataSource ds) {
    dataSource = ds;
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> create(String firstName, String lastName) {
    JdbcTemplate insert = <span class="hl-keyword">new</span> JdbcTemplate(dataSource);
    insert.update(<span class="hl-string" style="color:blue">"INSERT INTO PERSON (FIRSTNAME, LASTNAME) VALUES(?,?)"</span>,
        <span class="hl-keyword">new</span> Object[] { firstName, lastName });
  }

  <span class="hl-keyword">public</span> List&lt;Person&gt; select(String firstname, String lastname) {
    JdbcTemplate select = <span class="hl-keyword">new</span> JdbcTemplate(dataSource);
    <span class="hl-keyword">return</span> select
        .query(<span class="hl-string" style="color:blue">"select  FIRSTNAME, LASTNAME from PERSON where FIRSTNAME = ? AND LASTNAME= ?"</span>,
            <span class="hl-keyword">new</span> Object[] { firstname, lastname },
            <span class="hl-keyword">new</span> PersonRowMapper());
  }

  <span class="hl-keyword">public</span> List&lt;Person&gt; selectAll() {
    JdbcTemplate select = <span class="hl-keyword">new</span> JdbcTemplate(dataSource);
    <span class="hl-keyword">return</span> select.query(<span class="hl-string" style="color:blue">"select FIRSTNAME, LASTNAME from PERSON"</span>,
        <span class="hl-keyword">new</span> PersonRowMapper());
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> deleteAll() {
    JdbcTemplate delete = <span class="hl-keyword">new</span> JdbcTemplate(dataSource);
    delete.update(<span class="hl-string" style="color:blue">"DELETE from PERSON"</span>);
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> delete(String firstName, String lastName) {
    JdbcTemplate delete = <span class="hl-keyword">new</span> JdbcTemplate(dataSource);
    delete.update(<span class="hl-string" style="color:blue">"DELETE from PERSON where FIRSTNAME= ? AND LASTNAME = ?"</span>,
        <span class="hl-keyword">new</span> Object[] { firstName, lastName });
  }

} </pre><p>
