		<h1 class="firstheading"><a href="http://www.codingunit.com/c-tutorial-variables-and-constants" rel="bookmark" title="C Tutorial &#8211; variables and constants">
		C Tutorial &#8211; variables and constants</a></h1>

			<div class="entry">
				<div id="watermark">
					<p>In this C programming language tutorial we take a look at variables and constants.</p>
<h2>Variables</h2>
<p>If you declare a variable in C (later on we talk about how to do this), you ask the operating system for a piece of memory. This piece of memory you give a name and you can store something in that piece of memory (for later use). There are two basic kinds of variables in C which are numeric and character. <span id="more-142"></span></p>
<h2>Numeric variables</h2>
<p>Numeric variables can either be of the type integer (int) or of the type real (float). Integer (int) values are whole numbers (like 10 or -10). Real (float) values can have a decimal point in them. (Like 1.23 or -20.123).</p>
<h2>Character variables</h2>
<p>Character variables are letters of the alphabet, ASCII characters or numbers 0-9. If you declare a character variable you must always put the character between single quotes (like so &#8216;A&#8217; ). So remember a number without single quotes is not the same as a character with single quotes.</p>
<h2>Constants</h2>
<p>The difference between variables and constants is that variables can change their value at any time but constants can never change their value. (The constants value is lockedfor the duration of the program). Constants can be very useful, Pi for instance is a good example to declare as a constant.</p>
<h2>Data Types</h2>
<p>So you now know that there are three types of variables: numeric &#8211; integer, numeric-real and character. A variable has a type-name, a type and a range (minimum / maximum). In the following table you can see the type-name, type and range:</p>

<h2>Declaring</h2>
<p>So we now know different<br />
type-names and types of variables, but how do we declare them. Declaring a variable is very easy. First you have to declare the type-name. After the type-name you place the name of the variable. The name of a variable can be anything you like as long it includes only letters, underscores or numbers (However you cannot start the name with a number). But remember choose the names wisely. It is easier if a variable name reflects the use of that variable. (For instance: if you name a float PI, you always know what it means).</p>
<p>Now let&#8217;s declare some 	variables, a variable <em>MyIntegerVariable</em> and <em>MyCharacterVariable</em>:</p>
<div id="cl">
<pre><code>
	int main()
	{
		int MyIntegerVariable;
		int MyCharacterVariable;
		return 0;
	}
	</code></pre>
</div>
<p>It is possible to declare more than one variable at the same time:</p>
<div id="cl">
<pre><code>
	int main()
	{
		int Variable1, Variable2, Variable3;
		int abc, def, ghi;
		return 0;
	}
	</code></pre>
</div>
<p>To declare a constant is not much different then declaring a variable. The only difference is that you have the word const in front of it:</p>
<div id="cl">
<pre><code>
	int main()
	{
		const float PI = 3.14;
		char = 'A';
		return 0;
	}
	</code></pre>
</div>
<p><strong>Note:</strong> As you can see, you can assign a value with the equal sign during declaration.</p>
<h2>Signed and unsigned variables</h2>
<p>The difference between signed and unsigned variables is that signed variables can be either negative or<br />
positive but unsigned variables can only be positive. By using an unsigned variable you can increase the maximum positive range. When you declare a variable in the normal way it is automatically a signed variable. To declare an unsigned variable you just put the word unsigned before your variable declaration or signed for a signed variable although there is no reason to declare a variable as signed since they already are.</p>
<div id="cl">
<pre><code>
	int main()
	{
		unsigned int MyOnlyPositiveVar;
		signed int MyNegativeAndPositiveVar;
		int MyNegativeAndPositiveVar;
	}
	</code></pre>
</div>

<div id="cl">
<pre><code>
int main()
{
	int a, b;
	a = 1;
	b = a + 1;
	a = b - 1;
	return 0;
}

</code></pre>
</div>
<h2>Reading and printing</h2>
<p>Calculating something without reading input or printing something on the screen is not much fun. To read input from the keyboard we will use the command scanf. (How to print something to the screen we all ready<br />
know).</p>
<p>So let&#8217;s make a program that can do all these things:</p>
<div id="cl">
<pre><code>
#include&lt;stdio.h&gt;

int main()
{
	int inputvalue;

	scanf("%d", &amp;inputvalue);
	inputvalue = inputvalue * 10;
	printf("Ten times the input equals %d\n",inputvalue);
	return 0;
}

</code></pre>
</div>
<p><strong><strong>Note:</strong></strong> The input must be a whole number (integer).</p>
<p>The &amp; sign will be explained in a later tutorial. The %d is for reading or printing a decimal integer value (It is also possible to use %i). In the table below you can find the commands for other types:</p>
