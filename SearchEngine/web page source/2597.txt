                  <div>
                     <h2 class="title"><a name="jarfiles"></a>1.&nbsp;Defining and using Java libraries
                     </h2>
                  </div>
               </div>
            </div><a class="indexterm" name="d432514e51"></a><div class="section">
               <div class="titlepage">
                  <div>
                     <div>
                        <h3 class="title"><a name="jarfiles_definition"></a>1.1.&nbsp;What is a JAR file?
                        </h3>
                     </div>
                  </div>
               </div><a class="indexterm" name="d432514e57"></a><p>
                  A
                  JAR
                  file is a Java archive based on the pkzip file format. JAR
                  files are
                  the deployment format for Java. A
                  JAR can
                  contain Java classes
                  and other resources (icons, property files, etc.)
                  and
                  can be
                  executable.
                  
               </p>
               <p> You can
                  distribute your program in a jar file or you can use
                  existing java
                  code via jars by putting them into your classpath.
                  
               </p>
            </div>
            <div class="section">
               <div class="titlepage">
                  <div>
                     <div>
                        <h3 class="title"><a name="jarfiles_library"></a>1.2.&nbsp;Using Java libraries
                        </h3>
                     </div>
                  </div>
               </div>
               <p>
                  If you add a
                  JAR
                  file to your classpath, you can use its classes
                  in
                  your Java
                  application.
                  
               </p>
            </div>
         </div>
         <div class="section">
            <div class="titlepage">
               <div>
                  <div>
                     <h2 class="title"><a name="jarfiles_executable"></a>2.&nbsp;Executable JAR
                     </h2>
                  </div>
               </div>
            </div><a class="indexterm" name="d432514e72"></a><p>
               An executable
               JAR
               means the end-user can run the Java application
               without explicitly
               specifying the Java class which should be started.
               This is done via a
               <code class="filename">MANIFEST.MF</code>
               file which tells the JVM among other things which class
               contains the
               <code class="code">main()</code>
               method and the classpath.
               
            </p>
            <p>
               The following example shows one possible
               <code class="filename">MANIFEST.MF</code>
               file.
               
            </p>
            <p>
               
            </p><pre class="programlisting">Manifest-Version: <span class="hl-number">1.0</span> 
Main-Class: com.vogella.example.MyApp 
Class-Path:. lib/jcommon-<span class="hl-number">1.0</span>.6.jar lib/itext-<span class="hl-number">1.4</span>.6.jar </pre><p>
               
            </p>
            <p>
               An empty line is required at the end of the
               <code class="filename">MANIFEST.MF</code>
               file.
               
            </p>
            <p>
               You can create
               an executable JAR file via the following command.
               
            </p><pre class="programlisting"> jar -cvmf MANIFEST.MF app1.jar *.<span class="hl-keyword">class</span> </pre><p>
               
            </p>
            <p>
               Eclipse provides an export wizard to create a JAR file which can be
               started via
               <span class="guimenu">File</span> &#8594; <span class="guisubmenu">Export</span> &#8594; <span class="guisubmenu">Java</span> &#8594; <span class="guisubmenu">JAR file</span>
               
            </p>
            <p>
               
            </p>
