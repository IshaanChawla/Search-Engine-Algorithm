<h1>HTML - HyperText Markup Language</h1>

<p>Welcome to Tizag.com's HTML Tutorial! Here you will learn the basics of HyperText Markup Language (HTML), so that you may design your own web pages like the one you are viewing right now!</p>

<p>HTML is not a programming language, but rather a <b>markup</b> language. If you already know XML,  HTML will be a snap for you to learn. We urge you not to attempt to blow through this tutorial in one sitting. Instead, we recommend that you spend 15 minutes to an hour a day practicing HTML and then take a break to let the information settle in.  We aren't going anywhere!</p>

<h1>Preparation for HTML</h1>

<p>If you are new to HTML and haven't read through our <a HREF="../beginnerT/">Beginner's Tutorial</a>, please take a few minutes to complete that tutorial before moving on.</p>

<p>Creating an HTML document is easy.  To begin coding HTML, you need only two things: a <b>simple-text</b> editor, such as <b>Notepad</b>, and the dedication to follow this tutorial! Notepad is the most basic of simple-text editors, and you will probably code a fair amount of HTML with it in your early stages. Notepad++ is another popular favorite among web developers. These innovative text editors are specialized for writing simple code and they utilize color coding to help you write concise code.</p>

<h1>Brief HTML Background</h1>

<p>HTML hasn't been around for many years. The first web pages began in November 1990, and back then, there were little to no HTML standards to follow. As a result, a group called the <a HREF="http://www.w3.org/" TARGET="_blank"> World Wide Web Consortium</a> formed to set standards for coding HTML. We will base our teachings around these widely-accepted coding standards.</p>

<h1>Web Pages</h1>

<p>Here are some important facts about <i>why</i> web pages are so useful!</p>
<ul>
	<li>They are a low-cost and easy way to spread information to a large audience.</li>
	<li>The provide yet another medium you can use to market your business!</li>
	<li>They serve as a platform to let the world know about you!</li>
</ul>

<h1>Words to Know</h1>

<p>Throughout this tutorial, we will be using several terms that are unique to HTML. It is important for you to understand what these words mean, in the context of HTML. </p>

<ul>
	<li><b>Tag</b> - Used to tag or "mark-up" pieces of text. Once tagged, the text becomes HTML code to be interpreted by a web browser. Tags look
	like this: &lt;tag&gt;</li>
	<li><b>Element</b> - A complete tag, having an opening &lt;tag&gt; and a closing &lt;/tag&gt;.</li>
	<li><b>Attribute</b> - Used to modify the value of the HTML element. Elements
	will often have multiple attributes. </li>
</ul>

<p>For now, just understand that a <b>tag</b> a piece of code that acts as a label that a web browser interprets,  an <b>element</b> is a complete tag with an opening and closing tag, and an <b>attribute</b> is a property value that customizes or modifies an HTML element.</p>
