<td  id="content" colspan="7"><h1 class="content_heading">PrimeFaces Tutorial Series</h1>
<h1 class="subtitle">PrimeFaces with JSF2 and Eclipse</h1><!-- #BeginLibraryItem "/Library/JsfTrainingHeader.lbi" -->
<h2 class="content_subheading" align="center"><font color="#0000ff">Interested in live training from 
the author of these tutorials?
See the upcoming <a href="http://courses.coreservlets.com/public-courses/jsf2/">JSF 2.2</a> and 
<a href="http://courses.coreservlets.com/public-courses/primefaces/">PrimeFaces</a> training courses in Maryland,
co-sponsored by Johns Hopkins. Or, contact 
<a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a> for info on
customized JSF 2 or PrimeFaces courses onsite at <i>your</i> location. 
See  
<a href="http://courses.coreservlets.com/jsf-training.html">the JSF and PrimeFaces training page</a> for
various options for the onsite courses.
</font></h2><br/>
<!-- #EndLibraryItem --><!--
-->
<p>Following is a series of introductory tutorials on PrimeFaces, a popular third-party component library for JSF 2.
  This is intended as a beginner's tutorial, aimed at developers who already know
  <a href="http://www.coreservlets.com/JSF-Tutorial/jsf2/">JSF 2</a>, but who are new to PrimeFaces. It shows the most common
  usage of the most popular elements, but does not discuss every option or cover nearly all of the myriad PrimeFaces components.
  So, it is intended as a fast start for newcomers to PrimeFaces,
  and does <i>not</i> replace the PrimeFaces showcase and User's Guide for experienced PrimeFaces developers.
  Since each section includes exercises and exercise solutions, this can also be viewed as a self-paced introductory PrimeFaces training course.
  All the slides, source code, exercises, and exercise solutions are free for unrestricted use.
  Click on a section below to expand its content. The relatively few parts on IDE development and deployment use
  Eclipse, but of course none of the actual code is Eclipse-specific. These tutorials assume that you already know Java and at least the
  basics of JSF 2. If you don't already know JSF version 2, see <a href="http://www.coreservlets.com/JSF-Tutorial/jsf2/">the JSF 2 tutorial series</a>. If you don't already know the Java language, please see
  <a href="http://courses.coreservlets.com/Course-Materials/java.html">the Java programming tutorial series</a>.
  If you already have JSF or PrimeFaces experience, see the <a href="http://jobs.coreservlets.com/">JSF and
    PrimeFaces jobs page</a>.
</p>

<div class="accordion-div">
  <h1 class="book_part"><a class="accordionLink" href="#Tutorial-Intro" name="Tutorial-Intro"><b>Overview of the PrimeFaces Tutorial Series</b></a></h1>
<div> <!-- Accordion content -->
  <p>
    For most types of applications, JSF 2 is now the recommended way to build Java-based Web applications, and
    it is dramatically better than JSF 1 in almost every way, and is both more powerful and easier to use.
    However, JSF 2 does not include rich GUI elements; JSF leaves this task to the third-party component libraries.
    PrimeFaces is at least as powerful as the competing component libraries, and is significant simpler to get started with than most of them.
    Depending how you measure it, PrimeFaces now possibly the most popular choice, arguably more widely used than RichFaces, IceFaces, and Oracle ADF Faces.
    Click on a topic below to get the detailed tutorial for that topic, download the section's source code as an Eclipse project,
    see exercises, get the source code for the exercise solutions, or to simply run the sample apps from that section.
    Practicing is the key to learning, so I strongly recommend that you try out a few of the exercises in each section
    before you peek at the solutions.
  </p>
  <p>
    These tutorials are derived from
  <a href="http://courses.coreservlets.com/jsf-training.html">Marty Hall's world-renowned live JSF 2 and PrimeFaces training courses</a>.
    Customized courses on JSF and PrimeFaces (or JSF2 or PrimeFaces separately) are usually taught on-site at customer locations, but
  <a href="http://courses.coreservlets.com/public-courses/">Ajax, GWT, JSF 2.0, PrimeFaces, Spring, Hibernate, RESTful Web Services,
    Android, Hadoop, and Java 7 training courses at public venues</a> are periodically scheduled for people with too few developers
    for an onsite course. For descriptions of the various other courses that are
    available, please see the <a href="http://courses.coreservlets.com/">Java EE and Ajax training course page</a>.
    To inquire about a customized training course at <i>your</i> location, please contact Marty at
  <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>.
  </p>
  <p>If you find these free tutorials helpful, we would appreciate it if you would
