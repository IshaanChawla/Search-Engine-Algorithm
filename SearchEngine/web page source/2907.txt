		<h1 class="firstheading"><a href="http://www.codingunit.com/c-tutorial-number-of-seconds-in-decade-and-visaversa" rel="bookmark" title="C Tutorial &#8211; Number of Seconds in Decade and Visa-Versa">
		C Tutorial &#8211; Number of Seconds in Decade and Visa-Versa</a></h1>

			<div class="entry">
				<div id="watermark">
					<p>In a <a href="http://www.codingunit.com/c-tutorial-how-to-use-time-and-date-in-c" title="How to use time and date in C">previous C language tutorial post</a> the user &#8220;nikko manlangit&#8221; asked the following question: “How to convert seconds to decade or vice-versa?”. This C tutorial make use of time.h functions to answer his question.<span id="more-1195"></span></p>
<h2>Short Answer</h2>
<p>It all depends on how accurate you want to calculate the numbers?</p>
<p>You can calculate with the following numbers:<br />
A day has 24 hours x 60 minutes x 60 seconds = 86,400 seconds per day.<br />
So a year has about 365 x 86,400 = 31,536,000 seconds<br />
Decade has 10 years, so previous number times ten.</p>
<p>Or you take into account leap years and leap seconds (more accurate):<br />
An easy way to do this is using the tm structure (time.h).</p>
<h2>Convert Decade to Seconds</h2>
<p>Because we want a real world example we use the tm structure of time.h. Take a look at the following example:</p>
<div id="cl">
<pre><code>
#include&lt;stdio.h&gt;
#include&lt;stdlib.h&gt;
#include&lt;time.h&gt;
#include&lt;limits.h&gt;

int main(void) {
	
	time_t start_time;
	time_t end_time;
	time_t convert_back_example1;
	time_t convert_back_example2;
	struct tm start_str_time;
	struct tm end_str_time;
	
	// Fill our starting structure
	start_str_time.tm_sec = 0;
	start_str_time.tm_min = 0;
	start_str_time.tm_hour = 0;
	start_str_time.tm_mday = 1;
	start_str_time.tm_mon = 0;
	start_str_time.tm_year = 2010-1900;
	start_str_time.tm_isdst = 0;

        // Fill our end structure
	end_str_time.tm_sec = 61;
	end_str_time.tm_min = 59;
	end_str_time.tm_hour = 23;
	end_str_time.tm_mday = 31;
	end_str_time.tm_mon = 11;
	end_str_time.tm_year = 2019-1900;
	end_str_time.tm_isdst = 0;
	
	start_time = mktime(&#038;start_str_time);
	end_time = mktime(&#038;end_str_time);
	
	printf("The difference was %f seconds.\n", difftime(end_time, start_time));
	
	// - Seconds to date/time examples
        // Example one
	start_str_time.tm_sec += difftime(end_time, start_time);
	convert_back_example1 = mktime(&#038;start_str_time);
	printf("%s", asctime(localtime(&#038;convert_back_example1)));

	// Example two shows string to date / time conversion, since epoch time.
	const char diff_time[] = "315532801";
	// Ignore milliseconds
	convert_back_example2 = atoi(diff_time);
	printf("Since epoch time: %s", asctime(localtime(&#038;convert_back_example2)));

	return 0;
}

</code></pre>
</div>
<p>The program starts with some variable declaration of time_t and structures tm types. Then the first tm structure is filled with a date in the beginning of a year. The second tm structure is filled with a date that is a decade later and is at the end of that year. Then function mktime() used used to fix the broken-down time values, so we get a <a href="http://www.codingunit.com/c-reference-time-h-time-type-time_t">time_t</a> type. The last call of the first part is the function <a href="http://www.codingunit.com/c-reference-time-h-function-difftime">difftime()</a>, that returns the difference between two time_t values in seconds. So that&#8217;s the first part done. We have now broken down a decade into seconds. (And leap years and leap seconds are taken into account.)</p>
<h2>Seconds to Date/Time Example</h2>
<p>The second part of the program convert the seconds back into a readable date/time string. In example one we use the start time tm structure and add the seconds to .tm_sec. Then the <a href="http://www.codingunit.com/c-reference-time-h-function-mktime">mktime()</a> function is used again to create a time_t type. We then use the asctime() and <a href="http://www.codingunit.com/c-reference-time-h-function-localtime">localtime()</a> functions to print the date/time of newly calculated end time (a decade later).</p>
<h2>Seconds to Date/Time Example since Epoch</h2>
<p>In the second example we input a string representing the number of seconds in a decade. But this time we want to calculate using the epoch time. So we use <a href="http://www.codingunit.com/c-reference-stdlib-h-function-atoi-convert-a-string-to-an-integer">atoi()</a> function to convert the into an integer. Then again we use <a href="http://www.codingunit.com/c-reference-time-h-function-asctime">asctime()</a> and localtime() functions to print the date/time of newly calculated end time (a decade later) from epoch.</p>
<p>That&#8217;s all for this tutorial. As you can see you can use the time_t and tm structure to get a more accurate example when you want to do conversions. </p>
