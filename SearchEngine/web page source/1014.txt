<div style="color:#000080;font-family:Arial;font-size:10pt">
<p>The <b>String
</b>class provides two
methods that allow you to search a string for a specified character or substring:
<code>
<b>
• indexOf( ) </b>Searches
for the first occurrence of a character or substring.
<b>
<br>• lastIndexOf( ) </b>Searches
for the last occurrence of a character or substring.</code>
<p>These two methods are overloaded in several different ways. In
all cases, the methods return the index at which the character or substring was found,
or –1 on failure.</p>
<p>To search for the first occurrence of a character, use int indexOf(int <i>ch</i>)
<br>To search for the last occurrence of a character, use int lastIndexOf(int <i>ch</i>)</p>
<p>Here, <i>ch
</i>is the character being
sought. To search for the first or last occurrence of a substring, use</p>
<p>int indexOf(String <i>str</i>)
<br>int lastIndexOf(String <i>str</i>)</p>
<p>Here, <i>str
</i>specifies the substring.
<br>You can specify a starting point for the search using these
forms:</p>
<code>
int indexOf(int <i>ch</i>,
int <i>startIndex</i>)
<br>int lastIndexOf(int <i>ch</i>,
int <i>startIndex</i>)
<br>int indexOf(String <i>str</i>,
int <i>startIndex</i>)
<br>int lastIndexOf(String <i>str</i>,
int <i>startIndex</i>)
</code>
<p>
Here, <i>startIndex
</i>specifies the index at
which point the search begins. For <b>indexOf(
)</b>, the search runs from <i>startIndex
</i>to the end of the string.
For <b>lastIndexOf( )</b>,
the search runs from <i>startIndex
</i>to zero.
<br>The following example shows how to use the various index methods
to search inside of
<b>
String</b>s:</p>
<code>
// Demonstrate indexOf() and lastIndexOf().
<br>class indexOfDemo {
<br>public static void main(String args[]) {
 
String s = &quot;Now is the time for all good men &quot; +
<br>&quot;to come to the aid of their country.&quot;;
<br>System.out.println(s);
<br>System.out.println(&quot;indexOf(t) = &quot; +
<br>s.indexOf('t'));
<br>System.out.println(&quot;lastIndexOf(t) = &quot; +
<br>s.lastIndexOf('t'));
<br>System.out.println(&quot;indexOf(the) = &quot; +
<br>s.indexOf(&quot;the&quot;));
<br>System.out.println(&quot;lastIndexOf(the) = &quot; +
<br>s.lastIndexOf(&quot;the&quot;));
<br>System.out.println(&quot;indexOf(t, 10) = &quot; +
<br>s.indexOf('t', 10));
<br>System.out.println(&quot;lastIndexOf(t, 60) = &quot; +
<br>s.lastIndexOf('t', 60));
<br>System.out.println(&quot;indexOf(the, 10) = &quot; +
<br>s.indexOf(&quot;the&quot;, 10));
<br>System.out.println(&quot;lastIndexOf(the, 60) = &quot; +
<br>s.lastIndexOf(&quot;the&quot;, 60));<br/>
}
<br>}
</code>
<p>
Here is the output of this program:
<br>Now is the time for all good men to come to the aid of their country.</font></font></p>
<code>
indexOf(t) = 7
<br>lastIndexOf(t) = 65
<br>indexOf(the) = 7
<br>lastIndexOf(the) = 55
<br>indexOf(t, 10) = 11
<br>lastIndexOf(t, 60) = 55
<br>indexOf(the, 10) = 44
<br>lastIndexOf(the, 60) = 55</code>
