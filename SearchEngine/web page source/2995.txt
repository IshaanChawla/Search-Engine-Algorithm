    <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even" property="content:encoded"><p>C programming language allows programmer to create arrays of&nbsp;arrays known as multidimensional arrays. For example:</p>

<pre>
float a[2][6];</pre>

<p>Here, <var>a</var> is an array of two dimension, which is an example of multidimensional array.</p>
<p>For better understanding of multidimensional arrays, array elements of above example can be thinked of as below:</p>
<p>&nbsp;</p>
<p><img alt="Working of multidimensional arrays in c programming" class="figure" src="../sites/tutorial2program/files/C-multidimensional-arrays-working.jpg" style="height:116px; width:384px" title="C multidimensional arrays" /></p>
<h2>Initialization of Multidimensional Arrays</h2>
<p>In C, multidimensional arrays can be initialized in different number of ways.</p>

<pre>
int c[2][3]={{1,3,0}, {-1,5,9}};
                 OR
int c[][3]={{1,3,0}, {-1,5,9}};
                 OR 
int c[2][3]={1,3,0,-1,5,9};</pre>

<p><strong>Initialization Of three-dimensional Array</strong></p>

<pre>
double cprogram[3][2][4]={ 
{{-0.1, 0.22, 0.3, 4.3}, {2.3, 4.7, -0.9, 2}},
 {{0.9, 3.6, 4.5, 4}, {1.2, 2.4, 0.22, -1}},
 {{8.2, 3.12, 34.2, 0.1}, {2.1, 3.2, 4.3, -2.0}} 
};</pre>

<p>Suppose there is a multidimensional array <code>arr[i][j][k][m]</code>. Then this array can hold i*j*k*m numbers of data.</p>
<p>Similarly, the array of any dimension can be initialized in C programming.</p>
<h3>Example of Multidimensional Array In C</h3>
<p><strong>Write a C program to find sum of two matrix of order 2*2 using multidimensional arrays where, elements of matrix are entered by user.</strong></p>

<pre>
<code>
#include &lt;stdio.h&gt;
int main(){
   float a[2][2], b[2][2], c[2][2];
   int i,j;
   printf(&quot;Enter the elements of 1st matrix\n&quot;);
/* Reading two dimensional Array with the help of two for loop. If there was an array of &#39;n&#39; dimension, &#39;n&#39; numbers of loops are needed for inserting data to array.*/   
   for(i=0;i&lt;2;++i)      
       for(j=0;j&lt;2;++j){
       printf(&quot;Enter a%d%d: &quot;,i+1,j+1);
       scanf(&quot;%f&quot;,&amp;a[i][j]);
       }
   printf(&quot;Enter the elements of 2nd matrix\n&quot;);
   for(i=0;i&lt;2;++i)
       for(j=0;j&lt;2;++j){
       printf(&quot;Enter b%d%d: &quot;,i+1,j+1);
       scanf(&quot;%f&quot;,&amp;b[i][j]);
       }
   for(i=0;i&lt;2;++i)
       for(j=0;j&lt;2;++j){
/* Writing the elements of multidimensional array using loop. */
       c[i][j]=a[i][j]+b[i][j];  /* Sum of corresponding elements of two arrays. */
       }
   printf(&quot;\nSum Of Matrix:&quot;);
   for(i=0;i&lt;2;++i)
       for(j=0;j&lt;2;++j){
       printf(&quot;%.1f\t&quot;,c[i][j]);  
           if(j==1)             /* To display matrix sum in order. */
              printf(&quot;\n&quot;);
      }
return 0;
}
</code>
</pre>

<p><strong>Ouput</strong></p>

<pre>
<samp>Enter the elements of 1st matrix
Enter a11: 2;
Enter a12: 0.5;
Enter a21: -1.1;
Enter a22: 2;
Enter the elements of 2nd matrix
Enter b11: 0.2;
Enter b12: 0;
Enter b21: 0.23;
Enter b22: 23;

Sum Of Matrix:
2.2     0.5
-0.9    25.0</samp></pre>
