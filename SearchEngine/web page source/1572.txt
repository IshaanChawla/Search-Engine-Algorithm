<p></p>
<p>So what is &#8220;Hello, world!&#8221; exactly?  &#8220;Hello, world!&#8221; is a collection of sequential characters called a <strong>string</strong>.  In C++, we use strings to represent text such as names, addresses, words, and sentences.  String literals (such as &#8220;Hello, world!&#8221;) are placed between double quotes to identify them as a string.</p>
<p>Because strings are commonly used in programs, most modern languages include a built-in string data type.  C++ includes one, not as part of the core language, but as part of the standard library.</p>
<p><strong>std::string</strong></p>
<p>To use strings in C++, we first need to #include the &lt;string> header to bring in the declarations for std::string.  Once that is done, we can define variables of type std::string.</p>

<p>Hmmm, that isn&#8217;t right!  What happened?  It turns out that when using operator>> to extract a string from cin, operator>> only returns characters up to the first whitespace it encounters.  Any other characters are left inside cin, waiting for the next extraction.</p>
<p>So when we used operator>> to extract a string into variable name, only &#8220;John&#8221; was extracted, leaving &#8220;Doe&#8221; inside std::cin, waiting for the next extraction.  When we used operator>> again to extract a string into variable age, we got &#8220;Doe&#8221; instead of &#8220;23&#8221;.  If we had done a third extraction, we would have gotten &#8220;23&#8221;.</p>

Alex has 4 characters
</pre>
<p>Note that instead of asking for the string length as <code>length(myName)</code>, we say <code>myName.length()</code>.</p>
<p>The length function isn&#8217;t a normal standalone function like we&#8217;ve used up to this point -- it&#8217;s a special type of function that belongs to std::string called a member function. We&#8217;ll cover member functions, including how to write your own, in more detail later.</p>
<p><strong>Conclusion</strong></p>
<p>std::string is complex, leveraging many language features that we haven&#8217;t covered yet.  They also have a lot of other capabilities that we haven&#8217;t touched on here.  Fortunately, you don&#8217;t need to understand these to use them for simple tasks, like basic string input and output.  We encourage you to start experimenting with strings now, and we&#8217;ll cover additional string capabilities later.</p>

<p>1) Write a program that asks the user to enter their full name and their age.  As output, tell the user how many years they&#8217;ve lived for each letter in their name (for simplicity, count spaces as a letter).</p>
<p>Sample output:</p>
<pre>
Enter your full name: John Doe
Enter your age: 46
You've lived 5.75 years for each letter in your name.
</pre>
