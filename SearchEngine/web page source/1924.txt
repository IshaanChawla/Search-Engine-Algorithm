<td  id="content" colspan="7"><h1 class="content_heading">Android Programming Tutorials</h1>
<h1 class="subtitle">Developing Mobile Apps in Java</h1>

<h2 class="content_subheading" align="center"><font color="#0000ff">Interested in training from the author of these tutorials?
  See the upcoming <a href="http://courses.coreservlets.com/public-courses/android/">Android training course</a> in Maryland, co-sponsored
  by Johns Hopkins <i>Engineering for Professionals</i>. Or, 
  contact <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a> for info on
  customized Android courses at <i>your</i> location.</font></h2><br/>

<p>
  Following is a series of tutorials on Android programming. Since each section includes
  exercises and exercise solutions, this can also be viewed as a self-paced Android training course.
  Click on a section below to expand its content. These tutorials assume that you already know Java; they definitely move too fast
  for those without at least moderate prior Java experience. If you don't already know the Java language, please see
  <a href="http://courses.coreservlets.com/Course-Materials/java.html">the Java programming tutorial series</a>.
</p>


<div class="accordion-div">
  <h1 class="book_part"><a name="Overview"><b>Overview of the Android Tutorials</b></a></h1>
<div> <!-- Accordion content -->
  <p>
    Although Web applications are important and widely used, there are several reasons why native applications
    are sometimes preferable for mobile apps.
  </p>
  <ul>
    <li>You can use graphics and UI controls that are not available in browsers.</li>
    <li>You can customize the apps for small screen sizes.</li>
    <li>You can use resources (location, address book, phone, network, local database) that are
      unavailable in Web-based applications.</li>
  </ul>
  <p>
    Android is the preferred platform for building corporate apps since you can install Android apps from your company Web site, 
    rather than submitting them to the Apple App Store as required for the iPhone (unless all employees jailbreak their
    phones).
  </p>
  Click on a topic below to get the detailed tutorial for that topic, download the section's source code as an Eclipse project, 
  see exercises, get the source code for the exercise solutions, or to simply run the sample apps from that section.
  Practicing is the key to learning, so I strongly recommend that you try out a few of the exercises in each section
  before you peek at the solutions.
  </p>
  <p>
    These tutorials are derived from
  <a href="http://courses.coreservlets.com/android-training.html">Marty Hall's world-renowned live Android training courses</a>.
    Customized courses on Android are usually taught on-site at customer locations, but
  <a href="http://courses.coreservlets.com/public-courses/">servlet, JSP, 
    Ajax, GWT, JSF 2.0, Spring, Hibernate, RESTful Web Services, Android, and Java 6 training courses
    at public venues</a> are periodically scheduled for people with too few developers
    for an onsite course. For descriptions of the various other courses that are
    available, please see the <a href="http://courses.coreservlets.com/">Java EE and Ajax training course page</a>. 
    To inquire about a customized training course at <i>your</i> location, please contact Marty at 
  <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>.
  </p>
  <p>If you find these free tutorials helpful, we would appreciate it if you would
  <a href="http://courses.coreservlets.com/Course-Materials/link-to-us.html">link to us</a>. Send corrections or feedback on any tutorial to
  <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>. 
  </p>
</div> <!-- Accordion content -->

<h1 class="book_part"><a name="Intro"><b>Android Programming: Introduction and Overview</b></a></h1>
<div> <!-- Accordion content -->
  <p>This section gives a high-level introduction to developing for the Android platform.
  </p>
  <ul>
    <li><a href="http://courses.coreservlets.com/Course-Materials/pdf/android/Android-Overview.pdf" target="_blank">
      Tutorial section in PDF</a> (best for printing and saving).</li>
    <li><a href="https://www.slideshare.net/martyhall/android-tutorial-introduction-and-overview" target="_blank">
      Tutorial section on SlideShare</a> (preferred by some for online viewing).</li>  
    <li>Topics covered:
      <ul>
        <li>Motivation
          <ul>
            <li>Web Apps vs. Mobile Apps</li>
            <li>iPhone Apps vs. Android Apps</li>
            </ul>
          </li>
        <li>Books and references</li>
        </ul>
      </li>
    <li><a href="http://courses.coreservlets.com/Course-Materials/pdf/android/exercises/overview.pdf" target="_blank">
      Exercises to reinforce the concepts in this section.</a> In general, I strongly recommend that you try out
      at least a few of the exercises before moving on to later sections, but in this case (since we haven't
      covered how to get started yet), calling these "exercises" is a bit of an exaggeration.</li>
