                  <div>
                     <h2 class="title"><a name="overview"></a>1.&nbsp;Overview
                     </h2>
                  </div>
               </div>
            </div>
            <p> Java Architecture for XML Binding (JAXB) is a Java standard that
               defines how Java objects are
               converted from and to XML. It uses a
               standard set of mappings.
               
            </p>
            <p>
               JAXB defines an API for reading and
               writing Java objects to
               and
               from
               XML documents.
               
            </p>
            <p>As JAXB is defined via a specification, it is possible to use
               different implementations for this standard. JAXB
               defines a
               service
               provider
               which
               allows the
               selection of the JAXB
               implementation.
               
            </p>
            <p>
               It applies
               a lot of
               defaults thus
               making reading and
               writing
               of XML
               via
               Java
               relatively easy.
               
            </p>
         </div>

                     </h2>
                  </div>
               </div>
            </div>
            <p> Create a new Java project called "de.vogella.xml.jaxb".</p>
            <p> Create the following domain model with the JAXB annotations.
               
            </p>
            <p>
               
            </p><pre class="programlisting"><span class="hl-keyword">package</span> de.vogella.xml.jaxb.model;

<span class="hl-keyword">import</span> javax.xml.bind.annotation.XmlElement;
<span class="hl-keyword">import</span> javax.xml.bind.annotation.XmlRootElement;
<span class="hl-keyword">import</span> javax.xml.bind.annotation.XmlType;

<em><span class="hl-annotation" style="color: gray">@XmlRootElement(name = "book")</span></em>
<em class="hl-comment" style="color: #080">// If you want you can define the order in which the fields are written</em>
<em class="hl-comment" style="color: #080">// Optional</em>
<em><span class="hl-annotation" style="color: gray">@XmlType(propOrder = { "author", "name", "publisher", "isbn" })</span></em>
<span class="hl-keyword">public</span> <span class="hl-keyword">class</span> Book {

  <span class="hl-keyword">private</span> String name;
  <span class="hl-keyword">private</span> String author;
  <span class="hl-keyword">private</span> String publisher;
  <span class="hl-keyword">private</span> String isbn;

  <em class="hl-comment" style="color: #080">// If you like the variable name, e.g. "name", you can easily change this</em>
  <em class="hl-comment" style="color: #080">// name for your XML-Output:</em>
  <em><span class="hl-annotation" style="color: gray">@XmlElement(name = "title")</span></em>
  <span class="hl-keyword">public</span> String getName() {
    <span class="hl-keyword">return</span> name;
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> setName(String name) {
    <span class="hl-keyword">this</span>.name = name;
  }

  <span class="hl-keyword">public</span> String getAuthor() {
    <span class="hl-keyword">return</span> author;
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> setAuthor(String author) {
    <span class="hl-keyword">this</span>.author = author;
  }

  <span class="hl-keyword">public</span> String getPublisher() {
    <span class="hl-keyword">return</span> publisher;
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> setPublisher(String publisher) {
    <span class="hl-keyword">this</span>.publisher = publisher;
  }

  <span class="hl-keyword">public</span> String getIsbn() {
    <span class="hl-keyword">return</span> isbn;
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> setIsbn(String isbn) {
    <span class="hl-keyword">this</span>.isbn = isbn;
  }

} </pre><p>
               
            </p>
            <p>
               
            </p><pre class="programlisting"><span class="hl-keyword">package</span> de.vogella.xml.jaxb.model;

<span class="hl-keyword">import</span> java.util.ArrayList;

<span class="hl-keyword">import</span> javax.xml.bind.annotation.XmlElement;
<span class="hl-keyword">import</span> javax.xml.bind.annotation.XmlElementWrapper;
<span class="hl-keyword">import</span> javax.xml.bind.annotation.XmlRootElement;

<em class="hl-comment" style="color: #080">//This statement means that class "Bookstore.java" is the root-element of our example</em>
<em><span class="hl-annotation" style="color: gray">@XmlRootElement(namespace = "de.vogella.xml.jaxb.model")</span></em>
<span class="hl-keyword">public</span> <span class="hl-keyword">class</span> Bookstore {

  <em class="hl-comment" style="color: #080">// XmLElementWrapper generates a wrapper element around XML representation</em>
  <em><span class="hl-annotation" style="color: gray">@XmlElementWrapper(name = "bookList")</span></em>
  <em class="hl-comment" style="color: #080">// XmlElement sets the name of the entities</em>
  <em><span class="hl-annotation" style="color: gray">@XmlElement(name = "book")</span></em>
  <span class="hl-keyword">private</span> ArrayList&lt;Book&gt; bookList;
  <span class="hl-keyword">private</span> String name;
  <span class="hl-keyword">private</span> String location;

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> setBookList(ArrayList&lt;Book&gt; bookList) {
    <span class="hl-keyword">this</span>.bookList = bookList;
  }

  <span class="hl-keyword">public</span> ArrayList&lt;Book&gt; getBooksList() {
    <span class="hl-keyword">return</span> bookList;
  }

  <span class="hl-keyword">public</span> String getName() {
    <span class="hl-keyword">return</span> name;
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> setName(String name) {
    <span class="hl-keyword">this</span>.name = name;
  }

  <span class="hl-keyword">public</span> String getLocation() {
    <span class="hl-keyword">return</span> location;
  }

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> setLocation(String location) {
    <span class="hl-keyword">this</span>.location = location;
  }
} </pre><p>
               
            </p>
            <p> Create the following test program for writing and reading the
               XML file.
               
            </p>
            <p>
               
            </p><pre class="programlisting"><span class="hl-keyword">package</span> test;

<span class="hl-keyword">import</span> java.io.File;
<span class="hl-keyword">import</span> java.io.FileReader;
<span class="hl-keyword">import</span> java.io.IOException;
<span class="hl-keyword">import</span> java.util.ArrayList;

<span class="hl-keyword">import</span> javax.xml.bind.JAXBContext;
<span class="hl-keyword">import</span> javax.xml.bind.JAXBException;
<span class="hl-keyword">import</span> javax.xml.bind.Marshaller;
<span class="hl-keyword">import</span> javax.xml.bind.Unmarshaller;

<span class="hl-keyword">import</span> de.vogella.xml.jaxb.model.Book;
<span class="hl-keyword">import</span> de.vogella.xml.jaxb.model.Bookstore;

<span class="hl-keyword">public</span> <span class="hl-keyword">class</span> BookMain {

  <span class="hl-keyword">private</span> <span class="hl-keyword">static</span> <span class="hl-keyword">final</span> String BOOKSTORE_XML = <span class="hl-string" style="color:blue">"./bookstore-jaxb.xml"</span>;

  <span class="hl-keyword">public</span> <span class="hl-keyword">static</span> <span class="hl-keyword">void</span> main(String[] args) <span class="hl-keyword">throws</span> JAXBException, IOException {

    ArrayList&lt;Book&gt; bookList = <span class="hl-keyword">new</span> ArrayList&lt;Book&gt;();

    <em class="hl-comment" style="color: #080">// create books</em>
    Book book1 = <span class="hl-keyword">new</span> Book();
    book1.setIsbn(<span class="hl-string" style="color:blue">"978-0060554736"</span>);
    book1.setName(<span class="hl-string" style="color:blue">"The Game"</span>);
    book1.setAuthor(<span class="hl-string" style="color:blue">"Neil Strauss"</span>);
    book1.setPublisher(<span class="hl-string" style="color:blue">"Harpercollins"</span>);
    bookList.add(book1);

    Book book2 = <span class="hl-keyword">new</span> Book();
    book2.setIsbn(<span class="hl-string" style="color:blue">"978-3832180577"</span>);
    book2.setName(<span class="hl-string" style="color:blue">"Feuchtgebiete"</span>);
    book2.setAuthor(<span class="hl-string" style="color:blue">"Charlotte Roche"</span>);
    book2.setPublisher(<span class="hl-string" style="color:blue">"Dumont Buchverlag"</span>);
    bookList.add(book2);

    <em class="hl-comment" style="color: #080">// create bookstore, assigning book</em>
    Bookstore bookstore = <span class="hl-keyword">new</span> Bookstore();
    bookstore.setName(<span class="hl-string" style="color:blue">"Fraport Bookstore"</span>);
    bookstore.setLocation(<span class="hl-string" style="color:blue">"Frankfurt Airport"</span>);
    bookstore.setBookList(bookList);

    <em class="hl-comment" style="color: #080">// create JAXB context and instantiate marshaller</em>
    JAXBContext context = JAXBContext.newInstance(Bookstore.<span class="hl-keyword">class</span>);
    Marshaller m = context.createMarshaller();
    m.setProperty(Marshaller.JAXB_FORMATTED_OUTPUT, Boolean.TRUE);

    <em class="hl-comment" style="color: #080">// Write to System.out</em>
    m.marshal(bookstore, System.out);

    <em class="hl-comment" style="color: #080">// Write to File</em>
    m.marshal(bookstore, <span class="hl-keyword">new</span> File(BOOKSTORE_XML));

    <em class="hl-comment" style="color: #080">// get variables from our xml file, created before</em>
    System.out.println();
    System.out.println(<span class="hl-string" style="color:blue">"Output from our XML File: "</span>);
    Unmarshaller um = context.createUnmarshaller();
    Bookstore bookstore2 = (Bookstore) um.unmarshal(<span class="hl-keyword">new</span> FileReader(BOOKSTORE_XML));
    ArrayList&lt;Book&gt; list = bookstore2.getBooksList();
    <span class="hl-keyword">for</span> (Book book : list) {
      System.out.println(<span class="hl-string" style="color:blue">"Book: "</span> + book.getName() + <span class="hl-string" style="color:blue">" from "</span>
          + book.getAuthor());
    }
  }
} </pre><p>
               
            </p>
            <p> If you run the BookMain an XML file will be created from the input
               objects. Afterwards the file is read again and the objects are re-created
               based on the XML file.
               
            </p>
