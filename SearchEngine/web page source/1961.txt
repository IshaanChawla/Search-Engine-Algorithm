<td  id="content" colspan="7"><h1 class="content_heading">Hadoop Tutorial:<br/>
  Developing Big-Data Applications with Apache Hadoop</h1>

<h2 class="content_subheading" align="center"><font color="#0000ff">Interested in live training from the author of these tutorials?
  See the upcoming <a href="http://courses.coreservlets.com/public-courses/hadoop/">Hadoop training course</a> in Maryland, co-sponsored
  by Johns Hopkins <i>Engineering for Professionals</i>. Or, contact <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a> for info on
  customized Hadoop courses onsite at <i>your</i> location.</font></h2><br/>

<p>Following is an extensive series of tutorials on developing Big-Data Applications with Hadoop. Since each section includes
  exercises and exercise solutions, this can also be viewed as a self-paced Hadoop training course.
  All the slides, source code, exercises, and exercise solutions are free for unrestricted use.
  Click on a section below to expand its content. The relatively few parts on IDE development and deployment use
  Eclipse, but of course none of the actual code is Eclipse-specific. These tutorials assume that you already know Java; they definitely move too fast
  for those without at least moderate prior Java experience. If you don't already know the Java language, please see
  <a href="http://courses.coreservlets.com/Course-Materials/java.html">the Java programming tutorial series</a>.</p>
<p>
  For options for customized Hadoop training onsite at <i>your</i> organization, please see
  <a href="http://courses.coreservlets.com/hadoop-training.html">the Hadoop training course page</a> or
  email <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>.
</p>

<div class="accordion-div">
  <h1 class="book_part"><a class="accordionLink" href="#Tutorial-Intro" name="Tutorial-Intro"><b>Overview of the Hadoop Tutorial Series</b></a></h1>
<div> <!-- Accordion content -->
  <p>
    It is becoming increasingly common to have data sets that are too large to be handled by traditional databases, or by <i>any</i> technique
    that runs on a single computer or even a small cluster of computers. In the age of Big-Data, Hadoop has evolved as the library of choice for handling it.
    This tutorial gives a thorough introduction to Hadoop, along with many of the supporting libraries and packages. It also includes a free downloadable
    virtual machine that already has Hadoop installed and configured, so that you can quickly write code and test it out. 
    See the "Source Code and Virtual Machine" section at the bottom of this tutorial.
  </p>
  <p>
    These tutorials are written by Hadoop expert <a href="http://courses.coreservlets.com/dima-may.html">Dima May</a>, and are
    derived from
  <a href="http://courses.coreservlets.com/">the world-renowned coreservlets.com live training courses</a>.
    Customized courses on Hadoop are usually taught on-site at customer locations, but
  <a href="http://courses.coreservlets.com/public-courses/"> 
    JSF 2.2, PrimeFaces, Spring, Hibernate, RESTful Web Services, Android, Hadoop, Ajax/jQuery, 
    GWT, Java 7, and Java 8 training courses
    at public venues</a> are periodically scheduled for people with too few developers
    for an onsite course. For descriptions of the various other courses that are
    available, please see the <a href="http://courses.coreservlets.com/">Java EE and Ajax training course page</a>. 
    For options for customized Hadoop training onsite at <i>your</i> organization, please see
  <a href="http://courses.coreservlets.com/hadoop-training.html">the Hadoop training course page</a> or
    email <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>.
  </p>
  <p>If you find these free tutorials helpful, we would appreciate it if you would
  <a href="http://courses.coreservlets.com/Course-Materials/link-to-us.html">link to us</a>. 
  </p>
</div> <!-- Accordion content -->

<h1 class="book_part"><a class="accordionLink" href="#Devel-Environment" name="Devel-Environment"><b>The Development Environment</b></a></h1>
<div> <!-- Accordion content -->
  <p>This section walks you through setting up and using the development environment, starting and stopping Hadoop, and so forth.</p>
  <ul>
    <li><a href="http://courses.coreservlets.com/Course-Materials/pdf/hadoop/00-Overview_01-VM.pdf" target="_blank">
      Tutorial section in PDF</a> (best for printing and saving). Also see the
      <a href="http://courses.coreservlets.com/Course-Materials/pdf/hadoop/00-HadoopTrainingVM.pdf">VM download and installation guide</a>
      </li>
    <li><a href="http://www.slideshare.net/martyhall/hadoop-tutorial-setup-and-testing-of-development-environment" target="_blank">
      Tutorial section on SlideShare</a> (preferred by some for online viewing).</li>
