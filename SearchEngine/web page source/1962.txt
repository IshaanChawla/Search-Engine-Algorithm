<h1 class="content_heading">Jakarta Struts (Action Framework)</h1>
	<h1 class="subtitle">Tutorial and Training Materials</h1>

	<p>This tutorial is derived from <a href="http://courses.coreservlets.com/#Struts-Courses">
	  Marty Hall's world-renowned live Jakarta Struts training course</a>.
	  Note that the course (and this tutorial) now covers Struts 1.2.
	  The course is usually taught on-site at customer locations, but
  <a href="../public-courses/index.html">servlet, JSP, Struts, and JSF training courses
    at public venues</a> are periodically scheduled for people with too few developers
	  for an onsite course. For descriptions of the various other courses that are
	  available, please
	  see <a href="http://courses.coreservlets.com/">the servlet, JSP, Struts,
	    and JSF training course page</a>.
	  To inquire about a customized training course at <i>your</i> location, please
	  contact Marty at <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>.
	  Courses on servlets, JSP, JSF, Shale, Hibernate, and Java programming also available.</p>
<p>If you find these free tutorials helpful, we would appreciate it if you would
  <a href="link-to-us.html">link to us</a>.</p>

	<p><b>Available free for university faculty: </b>The original PowerPoint versions of the
	  tutorials are available free for university faculty in certain cases.
	  Please see <a href="http://courses.coreservlets.com/Course-Materials/instructor-materials.html">instructor usage rules</a>
	  for details.</p>
