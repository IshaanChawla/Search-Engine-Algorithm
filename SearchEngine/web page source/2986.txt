  <div class="content">
    <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even" property="content:encoded"><p>In C programming, goto statement is used for altering the normal sequence of program execution by transferring control to some other part of the program.</p><h3>Syntax of goto statement</h3>
<pre>goto label;
.............
.............
.............
label: 
statement;
</pre>
<p>In this syntax, label is an<a href="http://www.programiz.com/c-programming/c-keywords-identifier" title="C Programming identifier"> identifier</a>. When, the control of program reaches to goto statement, the control of the program will jump to the <code>label:</code> and executes the code below it.</p><p style="text-align: center; "><img alt="Working of goto statement in C programming" class="figure" src="../sites/tutorial2program/files/working-goto.jpg" title="C goto statement"></p><h3>Example of goto statement</h3>
<pre><code>
/*  C program to demonstrate the working of goto statement. */
/* This program calculates the average of numbers entered by user. */
/* If user enters negative number, it ignores that number and 
   calculates the average of number entered before it.*/

# include &lt;stdio.h&gt;
int main(){
   float num,average,sum;
   int i,n;
   printf("Maximum no. of inputs: ");
   scanf("%d",&amp;n);
   for(i=1;i&lt;=n;++i){
       printf("Enter n%d: ",i);
       scanf("%f",&amp;num);
       if(num&lt;0.0)
       goto jump;             /* control of the program moves to label jump */ 
       sum=sum+num;
}
jump:
  average=sum/(i-1);       
  printf("Average: %.2f",average);
  return 0;
}
</code></pre>
<p><strong>Output</strong></p>
<pre><samp>Maximum no. of inputs: 4
Enter n1: 1.5
Enter n2: 12.5
Enter n3: 7.2
Enter n4: -1
Average: 7.07
</samp></pre>
<p>Though goto statement is included in ANSI standard of C, use of goto statement should be reduced as much as possible in a program.</p><h3>Reasons to avoid goto statement</h3><p>Though, using goto statement give power to jump to any part of program, using goto statement makes the logic of the program complex and tangled. In modern programming, goto statement is considered a harmful construct and a bad programming practice.</p><p>The goto statement can be replaced in most of C program with the use of break and continue statements. In fact, any program in C programming can be perfectly written without the use of goto statement. All programmer should try to avoid goto statement as possible as they can.</p></div></div></div>
