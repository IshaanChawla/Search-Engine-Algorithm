<ul style="margin-left: 275px;">
  <li>The onsite courses are cheaper (no venue to rent, so your price is lower).</li>
  <li>The onsite courses are more convenient (you don't have to travel).</li>
  <li>The onsite courses are more effective (the topics and pace can be 
    customized to suit your needs and background).</li>
  <li>For clients in the Baltimore/Washington area, the onsite courses have
    a more flexible schedule (e.g., we can meet evenings or afternoons or one or two days per week instead of 
    on <i>n</i> consecutive days).</li>

  <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>
</p>

  However, many clients simply do not have enough developers for an onsite
  course, so we periodically offer open enrollment courses as well.
  These courses still feature Marty's lively and entertaining presentation
  style, lots of relevant hands-on exercises, coverage of best practices, 
  up-to-date topics and examples, and
  the chance to ask as many hard questions as you would like.
  And we still offer the same guarantee: if you are not completely satisfied with
  the course, we will refund the full cost. All 2014 courses are
  held at <a href="jhu-dorsey-travel-info.html">the JHU Dorsey Center</a>
  in Elkridge, Maryland (5 minutes from BWI airport) and are co-sponsored by 
  the Johns Hopkins University <i>Engineering for Professionals</i> program.</p>
<p>
  The following summarizes the schedule for the public (open enrollment) training courses for 2014.
  The spring 2015 course schedules will be announced later.
  <a href="http://courses.coreservlets.com/">Onsite courses</a>,
  however, can be arranged at your organization at any mutually agreeable time.</p>

    (<a href="jhu-dorsey-travel-info.html" itemprop="url"><span itemprop="name">Johns Hopkins Dorsey Center</span></a>)</span><!-- #EndLibraryItem --></legend>
  <p class="public_course_info">
    <time itemprop="startDate" dateTime="2014-08-12T08:30-05:00">August 25</time> &mdash; 
    <time itemprop="endDate" dateTime="2014-08-16T016:30-05:00">26, 2014</time><br/>
    Also evenings (4:00-8:00 pm) January 7/9 &amp; 14/16 2014
  </p>
  <p itemprop="description">
  <img src="../images/java-logo-small.png" align="left" alt="Java Logo"/>
    Java-day hands-on intro to the two most important new features to be added to Java in a decade: 
    lambdas and streams.
    Assumes that the students are already comfortable with earlier versions of Java. If not, take the
  <a href="java/index.html">general Java course</a> instead.
  <br/><br/>
    Covers motivation for lambdas, basic forms, type inferencing, using effectively final variables,
    @FunctionalInterface, method references, The java.util.function package (and Function, Predicate, Consumer, Supplier, and BinaryOperator), making streams, stream methods (especially forEach, filter, map, 
    and reduce), lazy evaluation, parallel streams, infinite streams, etc.
  </p>

    see <a href="java8/index.html">the Java 8 public training course page</a>.
    For customized onsite versions, see
  <a  href="../java-training.html">the onsite Java training page</a>.
  </p>
