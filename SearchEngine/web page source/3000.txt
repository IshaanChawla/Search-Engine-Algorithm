    <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even" property="content:encoded"><p>Pointers are the powerful feature of C and (C++) programming, which differs it from other popular programming languages like: java and Visual Basic.</p>
<p>Pointers are used in C program to access the memory and manipulate the address.</p>
<h3>Reference operator(&amp;)</h3>
<p>If <var>var</var> is a variable then, <code>&amp;var</code> is the address in memory.</p>

<pre>
<code>
/* Example to demonstrate use of reference operator in C programming. */
#include &lt;stdio.h&gt;
int main(){
  int var=5;
  printf(&quot;Value: %d\n&quot;,var);
  printf(&quot;Address: %d&quot;,&amp;var);  //Notice, the&nbsp;ampersand(&amp;) before var.
  return 0;
}
</code></pre>

<p><strong>Output</strong></p>

<pre>
<samp>Value: 5 
Address: 2686778</samp></pre>

<p><strong>Note:</strong> You may obtain different value of address while using this code.</p>
<p>In above source code, value 5 is stored in the memory location 2686778. <var>var</var> is just the name given to that location.</p>
<p>You, have already used reference operator in C program while using <code>scanf()</code> function.</p>

<pre>
scanf(&quot;%d&quot;,&amp;var);</pre>

<h3>Reference operator(*) and Pointer variables</h3>
<p>Pointer&nbsp;variable or simply pointer&nbsp;are the special types of variables that holds memory address rather than data, that is, a variable that holds address value is called a pointer variable or simply a pointer.</p>
<h3>Declaration of Pointer</h3>
<p>Dereference operator(*) are used for defining pointer variable</p>

<pre>
data_type* pointer_variable_name;
int* p;
</pre>

<p>Above statement defines, <var>p</var> as pointer variable of type <code>int</code>.</p>
<h3>Example To Demonstrate Working of Pointers</h3>

<pre>
<code>
/* Source code to demonstrate, handling of pointers in C program */
#include &lt;stdio.h&gt;
int main(){
   int* pc;
&nbsp;  int c;
   c=22;
   printf(&quot;Address of c:%d\n&quot;,&amp;c);
   printf(&quot;Value of c:%d\n\n&quot;,c);
   pc=&amp;c;
   printf(&quot;Address of pointer pc:%d\n&quot;,pc);
   printf(&quot;Content of pointer pc:%d\n\n&quot;,*pc);
   c=11;
   printf(&quot;Address of pointer pc:%d\n&quot;,pc);
   printf(&quot;Content of pointer pc:%d\n\n&quot;,*pc);
   *pc=2;
   printf(&quot;Address of c:%d\n&quot;,&amp;c);
   printf(&quot;Value of c:%d\n\n&quot;,c);
   return 0;
}
</code>
</pre>

<p><strong>Output</strong></p>

<pre>
<samp>Address of c: 2686784
Value of c: 22

Address of pointer pc: 2686784
Content of pointer pc: 22

Address of pointer pc: 2686784
Content of pointer pc: 11

Address of c: 2686784
Value of c: 2</samp></pre>

<p><img alt="Working of pointers in C programming" src="../sites/tutorial2program/files/pointers-in-c-programming.jpg" style="height:104px; width:532px" title="C pointers" /></p>
<p><strong>Explanation of program and figure</strong></p>
<ol><li>Code <code>int* pc;</code> creates a pointer <var>pc</var> and a code <code>int c;</code>&nbsp;creates normal variable&nbsp;<var>c</var>. Pointer <var>pc</var> points to some address and that address has garbage value. Similarly, variable <var>c</var> also has garbage value at this point.</li><li>Code<code>&nbsp;c=22;</code> makes the value of c equal to 22, i.e.,22 is stored in the memory location of variable <var>c</var>.</li><li>Code pc=&amp;c; makes pointer,&nbsp;point to address of c. Note that, <code>&amp;c</code> is the address of variable <var>c</var>&nbsp;(because <var>c</var> is normal variable) and <var>pc</var> is the address of <var>pc</var> (because pc is the pointer variable). Since the address of pc and address of c is same, <code>*pc</code>&nbsp;will be equal to the value of <var>c</var>.</li><li>Code <code>c=11;</code> makes the value of <var>c</var>, 11. Since, pointer <var>pc</var>&nbsp;is pointing to address of <var>c</var>. Value inside address&nbsp;<var>pc</var> will also be 11.</li><li>Code <code>*pc=2;</code> change the contents of the memory location pointed by pointer <var>pc</var> to change to 2. Since address of pointer <var>pc</var> is same as address of <var>c</var>, value of <var>c</var> also changes to 2.</li></ol>
<p>&nbsp;</p>
<h4>Commonly done mistakes in pointers</h4>
<p>Suppose, the programmer want pointer pc to point to the address of c. Then,</p>

<pre>
int c, *pc;
pc=c;  /* pc is address whereas, c is not an address. */
*pc=&amp;c; /* &amp;c is address whereas, *pc is not an address. */</pre>

<p>In both cases, pointer pc is not pointing to the address of c.</p></div></div></div>	
</div>
