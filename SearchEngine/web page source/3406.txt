<p>iPage is a shared web hosting solutions company. What does this mean? There are four main types of hosting solutions:</p>
<div style="position: relative; float: left; width: 230px; height: 400px; border-radius: 5px; background-color: #e9e9e9; margin-left: 0px;">
<p><img style="margin-left: 80px; margin-top: 10px;" alt="" src="http://www.mywebsitetraining.com/wp-content/uploads/2013/03/shared-icon.png" width="77" height="83" /></p>
<p style="font-size: 18px; color: #719f41; text-align: center; margin-left: 10px;">Shared Web Hosting</p>
<p style="font-size: 14px; margin-top: 20px; color: #666; margin-left: 10px; margin-right: 10px; text-align: center;">Shared web hosting means that a single machine is shared or houses multiple other customers, sometimes running hundreds in a single machine. This brings down the cost of hosting but does not give you direct access to the entire machine.</p>
</div>
<div style="position: relative; float: left; width: 230px; height: 400px; border-radius: 5px; background-color: #e9e9e9; margin-left: 10px;">
<p><img style="margin-left: 80px; margin-top: 10px;" alt="" src="http://www.mywebsitetraining.com/wp-content/uploads/2013/03/dedicated-hosting.png" width="71" height="83" /></p>
<p style="font-size: 18px; color: #719f41; text-align: center; margin-left: 10px;">Dedicated Web Hosting</p>
<p style="font-size: 14px; margin-top: 20px; color: #666; margin-left: 10px; margin-right: 10px; text-align: center;">Dedicated web hosting means that a single machine houses a single customer so that you have full access (and maintenance responsibilities) to that machine. Of course, this means a more expensive monthly tab for you.</p>
</div>
<div style="position: relative; float: left; width: 230px; height: 400px; border-radius: 5px; background-color: #e9e9e9; margin-left: 10px;">
<p><img style="margin-left: 40px; margin-top: 10px;" alt="" src="http://www.mywebsitetraining.com/wp-content/uploads/2013/03/vps-hosting1.png" width="152" height="83" /></p>
<p style="font-size: 18px; color: #719f41; text-align: center; margin-left: 10px;">VPS Hosting</p>
<p style="font-size: 14px; margin-top: 20px; color: #666; margin-left: 10px; margin-right: 10px; text-align: center;">VPS or virtual private server setups mean that your machine is partitioned into several virtual machines running independently from one another. Sort of like the dedicated web hosting but has a more flexible upgrade/downgrade feature.</p>
</div>
<div style="position: relative; float: left; width: 230px; height: 400px; border-radius: 5px; background-color: #e9e9e9; margin-left: 10px;">
<p><img style="margin-left: 70px; margin-top: 10px;" alt="" src="http://www.mywebsitetraining.com/wp-content/uploads/2013/03/cloud-hosting.png" width="86" height="83" /></p>
<p style="font-size: 18px; color: #719f41; text-align: center; margin-left: 10px;">Cloud Hosting</p>
<p style="font-size: 14px; margin-top: 20px; color: #666; margin-left: 10px; margin-right: 10px; text-align: center;">Cloud hosting is a configuration of several machines powering a cloud section. Sort of like a virtual server but you are being powered by various computers that are possibly in different locations around the world.</p>
</div>
