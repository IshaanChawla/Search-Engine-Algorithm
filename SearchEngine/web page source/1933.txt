<h1 class="content_heading">The Spring Framework: A Tutorial</h1>

<h2 class="content_subheading" align="center"><font color="#0000ff">Interested in training from the author of these tutorials?
  See the upcoming <a href="../public-courses/spring+hibernate/index.html">Spring &amp; Hibernate/JPA 
    courses in San Francisco and Maryland</a>, or contact <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a> for info 
  on customized courses at <i>your</i> location.</font></h2><br/>
<p>
  This tutorial is divided into two main sections. 
  <a href="#spring-basics">Section 1</a> is written by
  <a href="../about-instructor.html">Marty Hall</a> and
  provides an easy-to-understand tutorial for Spring beginners. If you are 
  new to Spring or care only about the core Spring capability of defining
  beans and dependency injection (inversion of control), this is the section for you.
  This section is used as part of <a href="../servlet+jsp-training.html">the
    coreservlets.com servlet and JSP training courses</a> or as a one-day standalone
  training class on Spring basics.
</p>
<p>
  <a href="#spring-in-detail">Section 2</a> is written by <a href="../spring-hibernate-instructors.html">Kyong Park</a>
  and provides a fast-moving tutorial that covers many different Spring capabilities and modules. If you
  are familiar with the basics of Spring, or are going to be using Spring extensively in your applications, 
  this is the section for you.  <a href="../spring-training.html">The detailed Spring training course</a>,
  on which this section is based is usually taught on-site at customer locations, but
  <a href="http://courses.coreservlets.com/public-courses/">various Java EE training courses
    at public venues</a> are periodically scheduled for people with too few developers for an onsite course. For descriptions of 	
  the various other courses that are available, please see 
  <a href="http://courses.coreservlets.com/">the J2EE training course page</a>. To inquire about a customized training course 
  at <i>your</i> location, please contact Marty at 
  <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>.
</p>
<p>If you find these free tutorials helpful, we would appreciate it if you would
  <a href="link-to-us.html">link to us</a>.</p>
