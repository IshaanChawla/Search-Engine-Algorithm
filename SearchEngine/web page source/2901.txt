		<h1 class="firstheading"><a href="http://www.codingunit.com/c-tutorial-copying-a-file-in-c" rel="bookmark" title="C Tutorial &#8211; Copying a File">
		C Tutorial &#8211; Copying a File</a></h1>

			<div class="entry">
				<div id="watermark">
					<p>In a previous tutorial we already talked about <a title="Deleting and Renaming a File" href="http://www.codingunit.com/c-tutorial-deleting-and-renaming-a-file"><strong>renaming and removing a file.</strong></a> In this C programming language tutorial we look at how to copy a file.</p>
<p>Let&#8217;s look at an example:<span id="more-223"></span></p>
<div id="cl">
<pre><code>

	#include&lt;stdio.h&gt;

	int copy_file(char *old_filename, char  *new_filename)
	{
		FILE  *ptr_old, *ptr_new;
		errno_t err = 0, err1 = 0;
		int  a;

		err = fopen_s(&amp;ptr_old, old_filename, "rb");
		err1 = fopen_s(&amp;ptr_new, new_filename, "wb");

		if(err != 0)
			return  -1;

		if(err1 != 0)
		{
			fclose(ptr_old);
			return  -1;
		}

		while(1)
		{
			a  =  fgetc(ptr_old);

			if(!feof(ptr_old))
				fputc(a, ptr_new);
			else
				break;
		}

		fclose(ptr_new);
		fclose(ptr_old);
		return  0;
	}

	int  main(void)
	{
		char  filename_src[101], filename_dest[101];

		printf("\nSource file: ");
		gets_s(filename_src, 100);

		printf("\nDestination filename: ");
		gets_s(filename_dest, 100);

		if(copy_file(filename_src, filename_dest) == 0)
			printf("Copy Successful\n");
		else
			fprintf(stderr, "Error during copy!");
	}

</code></pre>
</div>
<p>First we take a look at main before we look at the function copy_file().</p>
<h2>main()</h2>
<p>In main() we first make two buffers, a source and destination buffer, that each can hold 100 characters plus &#8216;\0&#8242;. Using the gets_s function (a more secure gets) the program waits for input from the user. The user must input a source file and a destination file. In the &#8220;if statement&#8221; the function copy_file() is called. The function is either successful or will fail. A message is printed accordingly.</p>
<h2>copy_file()</h2>
<p>In main the function copy_file() is called with the source filename and the destination file. The two files are then opened. The first file is opened in read only binary mode. The second file is opened in write binary mode. (If the file doesn&#8217;t exist the file is created.) For opening the file fopen_s() is used which is a more secure fopen().</p>
<p>After opening some error checking is done. If one of the two fopen_s() functions fails an -1 is returned.</p>
<p>In the &#8220;while statement&#8221; the content of the source file is read one byte at the time.<br />
This byte is written to the destination file. This will continue until the end of the file (EOF) is reached.<br />
If the end of the file is reached we break out with a break command.</p>
<p>Before the program ends both files are closed using fclose() function.</p>
<h2>Conclusion</h2>
<p>Make sure you close the files and always use secure functions when possible.</p>
<p>That is all for this tutorial!</p>
