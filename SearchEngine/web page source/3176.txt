<h1>JavaScript Tutorial</h1>
<p>Javascript--what the heck is it?  Is it a really difficult programming language that casual web designers should be afraid of?  What is it used for? Hopefully we'll
be able to answer these questions for you and more in this tutorial</p>
<div align='center'>

<p>JavaScript has been around for several years now, in many different flavors.  The main benefit of Javascript is to add additional interaction between the website
and its visitors with just a little extra work by the web developer. Javascript allows industrious web masters to get more out of their website than HTML and CSS can provide.</p>
<p>By definition, JavaScript is a client-side scripting language.  This means the web surfer's browser will be running the script. The opposite of client-side is server-side, which occurs in a language like PHP. PHP scripts are run by the web hosting server.</p>
<p>There are many uses (and abuses!) for the powerful JavaScript language.  Here are a few things that you may or may not have seen in your web surfing days:</p>
<ul>
	<li>Clocks</li>
	<li>Mouse Trailers (an animation that follows your mouse when you surf a site)</li>
	<li>Drop Down Menus</li>
	<li>Alert Messages</li>
	<li>Popup Windows</li>
	<li><a HREF="http://www.tizag.com/htmlT/forms.php">HTML Form</a> Data Validation</li>
</ul>
