<p><em><span class="style11">(Identifiers, keywords, literals, white spaces and comments)</span></em></p>
<p>This part of the java tutorial teaches you the <strong>basic language elements</strong> and syntax for the java programming language. Once you get these basic language concepts you can continue with the other object oriented programming language concepts.</p>
<h1 class="style1"><span style="font-weight: 400;">Keywords</span></h1>
<p>There are certain words with a specific meaning in java which tell (help) the compiler what the program is supposed to do. These Keywords cannot be used as variable names, class names, or method names. Keywords in java are case sensitive, all characters being lower case.</p>
<p>Keywords are reserved words that are predefined in the language; see the table below (Taken from Sun Java Site). All the keywords are in lowercase.</p>
<pre class="wp-code-highlight prettyprint">
	abstract    default    if            private      this
	boolean     do         implements    protected    throw
	break       double     import        public       throws
	byte        else       instanceof    return       transient
	case        extends    int           short        try
	catch       final      interface     static       void
	char        finally    long          strictfp     volatile
	class       float      native        super        while
	const       for        new           switch
	continue    goto       package       synchronized
</pre>
<p><span class="style4">Keywords are marked in yellow as shown in the sample code below </span></p>
<p>/** This class is a Hello World Program used to introduce<br />
the Java Language*/<br />
public class HelloWorld {<br />
public static void main(String[] args) {<br />
System.out.println(&#8220;Hello World&#8221;); //Prints output to console</p>
<p>}<br />
}</p>
<p><span class="style31">For more information on different Keywords</span> &#8211; <strong class="text2"><a onclick="javascript:pageTracker._trackPageview('/downloads/basic-language-elements/../Java-keywords.doc');"  href="../Java-keywords.doc">Java Keywords</a></strong></p>
<p><span class="style4">Some Tricky Observations:</span> The words virtual, ifdef, typedef, friend, struct and union are all words related to</p>
<p>the C programming language. const and goto are Java keywords. The word finalize is the name of a method</p>
<p>of the Object class and hence not a keyword. enum and label are not keywords.</p>
<h1 class="style1"><span style="font-weight: 400;">Comments</span></h1>
<p><a onclick="javascript:pageTracker._trackPageview('/outgoing/www.wideskills.com');"  href="http://www.wideskills.com"><img class="aligncenter size-full wp-image-572" alt="wideskills" src="http://www.javabeginner.com/wp-content/uploads/2015/02/wideskills-3.png" width="499" height="154" /></a></p>
<p>Comments are descriptions that are added to a program to make code easier to understand. The compiler ignores comments and hence its only for documentation of the program.</p>
<p class="style4">Java supports three comment styles.</p>
<p><em>Block style</em> comments begin with /* and terminate with */ that spans multiple lines.</p>
<p><em>Line style</em> comments begin with // and terminate at the end of the line. (Shown in the above program)</p>
<p><em>Documentation style</em> comments begin with /** and terminate with */ that spans multiple lines. They are generally created using the automatic documentation generation tool, such as javadoc. (Shown in the above program)</p>
<p>name of this compiled file is comprised of the name of the class with .class as an extension.</p>
<h2><span style="font-weight: 400;"> <span class="style1">Variable, Identifiers and Data Types</span></span></h2>
<p><strong>Variables </strong>are used for data that change during program execution. All variables have a name, a type, and a scope. The programmer assigns the names to variables, known as <strong>identifiers</strong>. An Identifier must be unique within a scope of the Java program. Variables have a <strong>data type</strong>, that indicates the kind of value they can store. Variables declared inside of a block or method are called <span class="style4">local variables</span>; They are not automatically initialized. The compiler will generate an error as a result of the attempt to access the local variables before a value has been assigned.</p>
<pre class="wp-code-highlight prettyprint">
public class localVariableEx {
  public static int a;
  public static void main(String[] args) {
    int b;
    System.out.println(&quot;a : &quot;+a);
    System.out.println(&quot;b : &quot;+b); 	//Compilation error
}}
</pre>
<p><strong>Note</strong> in the above example, a compilation error results in where the variable is tried to be accessed and not at the place where its declared without any value.</p>
<p>The <strong>data type</strong> indicates the attributes of the variable, such as the range of values that can be stored and the operators that can be used to manipulate the variable. Java has four main primitive data types built into the language. You can also create your own composite data types.</p>
<p>Java has four main primitive data types built into the language. We can also create our own data types.</p>
<div>
<ul>
<li><strong><em>Integer:</em></strong> byte, short, int, and long.</li>
<li><em><strong>Floating Point:</strong></em> float and double</li>
<li><em><strong>Character:</strong></em> char</li>
<li><em><strong>Boolean:</strong></em> variable with a value of true or false.</li>
</ul>

<p>When we declare a variable we assign it an identifier and a data type.</p>
<p class="style4">For Example</p>
<p>String message = &#8220;hello world&#8221;</p>
<p>In the above statement, String is the <strong>data type</strong> for the <strong>identifier</strong> message. If you don&#8217;t specify a value when the variable is declared, it will be assigned the default value for its data type.</p>
<p class="style4"><em>Identifier Naming Rules</em></p>
<ul>
<li>Can consist of upper and lower case letters, digits, dollar sign ($) and the underscore ( _ ) character.</li>
<li>Must begin with a letter, dollar sign, or an underscore</li>
<li>Are case sensitive</li>
<li>Keywords cannot be used as identifiers</li>
<li>Within a given section of your program or scope, each user defined item must have a unique identifier</li>
<li>Can be of any length.</li>
</ul>
<h3 class="style1"><span style="font-weight: 400;">Classes</span></h3>
<p>A class is nothing but a blueprint for creating different objects which defines its properties and behaviors. An object exhibits the properties and behaviors defined by its class. A class can contain fields and methods to describe the behavior of an object. Methods are nothing but members of a class that provide a service for an object or perform some business logic.</p>
<p><a href="http://americanportal.com/jobs.php" target="_top"><br />
</a></p>
<h3><span style="font-weight: 400;"> <span class="style1">Objects</span></span></h3>
<p>An object is an instance of a class created using a new operator. The new operator returns a reference to a new instance of a class. This reference can be assigned to a reference variable of the class. The process of creating objects from a class is called instantiation. An object reference provides a handle to an object that is created and stored in memory. In Java, objects can only be manipulated via references, which can be stored in variables.</p>
<h3><span style="font-weight: 400;"> <span class="style1">Interface</span></span></h3>
<p>An Interface is a contract in the form of collection of method and constant declarations. When a class implements an interface, it promises to implement all of the methods declared in that interface.</p>
<h3><span style="font-weight: 400;"> <span class="style1">Instance Members</span></span></h3>
<p>Each object created will have its own copies of the fields defined in its class called instance variables which represent an objects state. The methods of an object define its behaviour called instance methods. Instance variables and instance methods, which belong to objects, are collectively called instance members. The dot &#8216;.&#8217; notation with a object reference is used to access Instance Members.</p>
<h3><span style="font-weight: 400;"> <span class="style1">Static Members</span></span></h3>
<p>Static members are those that belong to a class as a whole and not to a particular instance (object). A static variable is initialized when the class is loaded. Similarly, a class can have static methods. Static variables and static methods are collectively known as static members, and are declared with a keyword static. Static members in the class can be accessed either by using the class name or by using the object reference, but instance members can only be accessed via object references.</p>
<p class="style4">Below is a program showing the various parts of the basic language syntax that were discussed above.</p>
<pre class="wp-code-highlight prettyprint">
/** Comment
 * Displays &quot;Hello World!&quot; to the standard output. 

 */
public class HelloWorld {
      String output = &quot;&quot;;
      static HelloWorld helloObj;  //Line 1 

      public HelloWorld(){
            output = &quot;Hello World&quot;;
      } 

      public String printMessage(){
            return output;
      } 

      public static void main (String args[]) {
            helloObj = new HelloWorld();  //Line 2
            System.out.println(helloObj.printMessage());
  }

}
</pre>
<p><strong>Class Name:</strong> HelloWorld<br />
<strong>Object Reference:</strong> helloObj (in Line 1)<br />
<strong>Object Created:</strong> helloObj (In Line 2)<br />
<strong>Member Function:</strong> printMessage<br />
<strong>Field:</strong> output (String)<br />
<strong>Static Member: </strong>helloObj<br />
<strong>Instance Member :</strong> output (String)</p>
