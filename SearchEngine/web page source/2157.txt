<h1 id="Apache_MyFaces">Apache MyFaces</h1>
<span class="anchor" id="line-5"></span><span class="anchor" id="line-6"></span><p class="line867">JavaServer(tm) Faces (JSF) is a web application framework specification (JSR252, JSR127, JSR314) based around the model-view-controller pattern. <span class="anchor" id="line-7"></span><span class="anchor" id="line-8"></span><p class="line862">The Apache MyFaces project provides both an implementation of the JSF specification ("MyFaces Core"), and a set of add-on libraries that work on any implementation of the specification (including, of course, Apache MyFaces Core). <span class="anchor" id="line-9"></span><span class="anchor" id="line-10"></span><p class="line862">JSF supports the development of highly interactive applications where the system interacting with the user is separated from the system that contains the presentation logic; web-browser/web-server is one example. JSF has a "component-based" architecture, and therefore is more like "Swing for the web" than an HTML publishing framework. People familiar with Swing, Gnome, KDE, MFC, VisualBasic, Delphi etc should feel right at home developing JSF applications. <span class="anchor" id="line-11"></span><span class="anchor" id="line-12"></span><p class="line862">JSF is quite similar to the <a class="http" href="http://jakarta.apache.org/tapestry">Tapestry framework</a> as that is also a component-based framework. JSF can also fill the same "niche" as the well-known <a class="http" href="http://jakarta.apache.org/struts/">Struts framework</a>, but has a rather different approach. <span class="anchor" id="line-13"></span><span class="anchor" id="line-14"></span><p class="line862">The <a class="http" href="http://java.sun.com/j2ee/javaserverfaces/">Sun JavaServer Faces(tm) Page</a> has information about the <a class="http" href="http://www.jcp.org/en/jsr/detail?id=252">JSF-1.2</a> (and older <a class="http" href="http://www.jcp.org/en/jsr/detail?id=127">JSF-1.1</a>) specification. They also provide a useful Tutorial there. MyFaces Core provides a 100% compliant implementation of these specifications. <span class="anchor" id="line-15"></span><span class="anchor" id="line-16"></span><p class="line862">The Apache MyFaces family of projects includes the following libraries that can be used with any JSF implementation to add extra features and flexibility: <span class="anchor" id="line-17"></span><ul><li><p class="line862">Tomahawk (<a href="/myfaces/Tomahawk">Tomahawk</a>),  <span class="anchor" id="line-18"></span></li><li><p class="line862">Tobago (<a href="/myfaces/Tobago">Tobago</a>), <span class="anchor" id="line-19"></span></li><li><p class="line862">Trinidad (<a href="/myfaces/Trinidad">Trinidad</a>), <span class="anchor" id="line-20"></span></li><li><p class="line862">Orchestra (<a href="/myfaces/Orchestra">Orchestra</a>), <span class="anchor" id="line-21"></span></li><li><p class="line862">Portlet Bridge (<a href="/myfaces/PortletBridge">PortletBridge</a>), <span class="anchor" id="line-22"></span></li><li><p class="line862">Extensions (<a href="/myfaces/Extensions">Extensions</a>), <span class="anchor" id="line-23"></span></li><li><p class="line862">Dojo facelets (unreleased + in progress, see <a class="http" href="http://www.j4fry.org/dojoFacelets.shtml">http://www.j4fry.org/dojoFacelets.shtml</a>) <span class="anchor" id="line-24"></span><span class="anchor" id="line-25"></span></li></ul><p class="line862">You can find further information about the MyFaces subprojects on the official site at <a class="http" href="http://myfaces.apache.org">http://myfaces.apache.org</a>. <span class="anchor" id="line-26"></span><span class="anchor" id="line-27"></span><p class="line862">See the MyFaces Tomahawk and Sandbox examples in action: <span class="anchor" id="line-28"></span><ul><li><p class="line891"><a class="http" href="http://www.irian.at/myfaces.jsf">http://www.irian.at/myfaces.jsf</a> <span class="anchor" id="line-29"></span><span class="anchor" id="line-30"></span></li></ul><p class="line862">See a MyFaces real world use case at JavaWorld <span class="anchor" id="line-31"></span><ul><li><p class="line891"><a class="http" href="http://www.javaworld.com/javaworld/jw-07-2006/jw-0731-myfaces.html">http://www.javaworld.com/javaworld/jw-07-2006/jw-0731-myfaces.html</a>. <span class="anchor" id="line-32"></span><span class="anchor" id="line-33"></span></li></ul><p class="line874">See the dojoFacelets in action: <span class="anchor" id="line-34"></span><ul><li><p class="line891"><a class="http" href="http://www.j4fry.org/JSFExamples20/page/dojo.faces">http://www.j4fry.org/JSFExamples20/page/dojo.faces</a> <span class="anchor" id="line-35"></span><span class="anchor" id="line-36"></span></li></ul><p class="line862">If your company uses MyFaces, please add your company here: <a href="/myfaces/Companies_Using_MyFaces">Companies_Using_MyFaces</a>. <span
