<div style="color:#000080;font-family:Arial;font-size:10pt">
<p>You can extract a substring using <b>substring(
)</b>. It has two forms. The
first is
<code>String substring(int <i>startIndex</i>)</code>
<p>Here, <i>startIndex
</i>specifies the index at
which the substring will begin. This form returns a copy of the substring that begins at <i>startIndex
</i>and runs to the end of
the invoking string.</p>
<p>The second form of <b>substring(
) </b>allows you to specify
both the beginning and ending index of the substring:</p>
<code>String substring(int <i>startIndex</i>,
int <i>endIndex</i>)</p>
</code><p>Here, <i>startIndex
</i>specifies the beginning
index, and <i>endIndex
</i>specifies the stopping
point. The string returned contains all the characters from the
beginning index, up to, but not including, the ending index. The following program uses <b>substring(
) </b>to replace all
instances of one substring with another within a string:</p>
<code>
  <p>// Substring replacement.
<br>class StringReplace {
<br>public static void main(String args[]) {
<br>String org = &quot;This is a test. This is, too.&quot;;
<br>String search = &quot;is&quot;;
<br>String sub = &quot;was&quot;;
<br>String result = &quot;&quot;;
<br>int i;
<br>do { // replace all matching substrings
<br>System.out.println(org);
<br>i = org.indexOf(search);
<br>if(i != -1) {
<br>result = org.substring(0, i);
<br>result = result + sub;
<br>result = result + org.substring(i + search.length());
<br>org = result;
<br>}
<br>} while(i != -1);
<br>}
<br>}</p>
</code>
<p>The output from this program is shown here:</p>
<code>This is a test. This is, too.
<br>Thwas is a test. This is, too.
<br>Thwas was a test. This is, too.
<br>Thwas was a test. Thwas is, too.
<br>Thwas was a test. Thwas was, too.</code>
