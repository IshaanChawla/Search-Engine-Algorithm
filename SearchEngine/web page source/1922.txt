<p>This tutorial covers JSF 1.x with Apache MyFaces, but the
  <a href="../jsf2/index.html">the JSF 2.0 tutorial</a> covers JSF 2 with the free Mojarra JSF2 implementation. 
  <b><font color="red">This JSF 1 tutorial
    is left online only for those maintaining existing projects: <a href="../jsf2/index.html">JSF 2</a> is both simpler and more powerful than JSF 1, and is
    strongly preferred for all new projects.</font></b>
  Click on a topic below to get the detailed tutorial,
  download the source code, or try out exercises on the topic.
</p>
<p>
  This tutorial is derived from
  <a href="http://courses.coreservlets.com/jsf-training.html">Marty Hall's world-renowned live JSF training courses</a>.
  Courses on JSF 1.x or 2.0 are usually taught on-site at customer locations, but
  <a href="http://courses.coreservlets.com/public-courses/">Java 7, servlet, JSP,
    Ajax (with jQuery), GWT, Spring, Hibernate, Web Services (SOAP and REST), Hadoop, JSF 1.x, JSF 2.0, and PrimeFaces training courses
    at public venues</a> are periodically scheduled for people with too few developers
  for an onsite course. For descriptions of the various other courses that are
  available, please see the <a href="http://courses.coreservlets.com/">Java EE and Ajax training course page</a>.
  To inquire about a customized training course at your location, please contact Marty at
  <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a>.
</p>

<div class="accordion-div">
  <h1 class="book_part"><a name="Review"><b>Servlets and JSP: A Review of the Basics</b></a></h1>
<div> <!-- Accordion content -->
  <p>This section provides a quick review of the basics of servlets and JSP, as a review
    before moving on to the JSF-specific topics.
    Please note that this section is not detailed enough for those who have never done server-side
    programming in Java; it is intended as a review for those who have done it before, but are rusty.
    If you are new to building Web apps in Java, please see <a href="http://courses.coreservlets.com/Course-Materials/csajsp2.html">
      the beginning and intermediate servlet and JSP tutorials</a>. </p>
  <ul>
    <li><a href="http://courses.coreservlets.com/Course-Materials/pdf/ajax/Servlet+JSP-Review.pdf" target="_blank">This tutorial section in PDF.</a></li>
    <li><a href="http://courses.coreservlets.com/Course-Materials/ajax-code/review.zip">Source code of examples in this section.</a>
      Bundled as an <a href="http://courses.coreservlets.com/Course-Materials/eclipse-code.html">Eclipse project</a>.</li>
    <li><a href="http://courses.coreservlets.com/Course-Materials/pdf/msajsp/exercises/exercises-review.pdf" target="_blank">Exercises to reinforce the concepts in this section.</a></li>
    <li><a href="http://courses.coreservlets.com/Course-Materials/ajax-code/review-exercises.zip">Exercise solutions.</a></li>
    <li>Topics covered:
      <ul>
        <li>What servlets are all about</li>
        <li>Servlet basics</li>
        <li>Creating and deploying projects in Eclipse. Also see
          <a href="http://www.coreservlets.com/Apache-Tomcat-Tutorial/eclipse.html">
            the Tomcat and Eclipse tutorial</a>.</li>
        <li>Creating forms and reading form data</li>
