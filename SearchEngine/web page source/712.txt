    <td>Returns the string value of the argument. The argument

      Result: &quot;314&quot;</p></td>
  </tr>
  <tr>

	Result: (84, 104, 233, 114, 232, 115, 101)</p></td>

	<p>Example: compare('ghi', 'ghi')<br>
	Result: 0</p></td>
    </tr>

	<p>Example: string-join(('We', 'are', 'having', 'fun!'))<br>
