<h1 class="h1">C Programming Language Tutorial</h1>

<p><b>C language</b> Tutorial with programming approach for beginners and professionals, helps you to understand the C language tutorial easily. Our C tutorial explains each topic with programs.</p>
<p>The C Language is developed for creating system applications that direct interacts to the hardware devices such as drivers, kernals etc.</p>
<p>C programming is considered as the base for other programming languages, that is why it is known as mother language.</p>
<p>

<p>It can be defined by following ways:</p>
<ol>
<li>Mother language</li>
<li>System programming language</li>
<li>Procedure-oriented programming language</li>
<li>Structured programming language</li>
<li>Mid level programming language</li>
</ol>
<hr/>
<h2 class="h2">1) C as a mother language</h2>
<p>C language is considered as the mother language of all the modern languages because <b>most of the compilers, JVMs, Kernals etc. are written in C language</b> and most of languages follows c syntax e.g. C++, Java etc.</p>
<p>It provides the core concepts like array, functions, file handling etc. that is being used in many languages like C++, java, C# etc.</p>

<hr/>
<h2 class="h2">2) C as a system programming language</h2>
<p>A system programming language is used to create system softwares. C language is a system programming language because it <b>can be used to do low level programming (e.g. driver and kernel)</b>. It is generally used to create hardware devices, OS, drivers, kernels etc.  For example, linux kernel is written in C.</p>
<p>It can?t be used in internet programming like java, .net, php etc.</p>

<hr/>
<h2 class="h2">3) C as a procedural language</h2>
<p>A procedure is known as function, method, routine, subroutine etc. A procedural language <b>specifies a series of steps or procedures for the program to solve the problem</b>.</p>
<p>A procedural language breaks the program into functions, data structures etc.</p>
<p>C is a procedural language. In C, variables and function prototypes must be declared before being used.</p>

<hr/>
<h2 class="h2">4) C as a structured programming language</h2>
<p>A structured programming language is a subset of procedural language. <b>Structure means to break a program into parts or blocks</b> so that it may be easy to understand. </p>
<p>In C language, we break the program into parts using functions. It makes the program easier to understand and modify.</p>

<hr/>
<h2 class="h2">5) C as a mid-level programming language</h2>
<p>C is considered as a middle level language because it <b>supports the feature of both low-level and high level language</b>. C language program is converted into assembly code, supports pointer arithmetic (low level), but it is machine independent (feature of high level).</p>
<p><b>Low level language</b> is specific to one machine i.e. machine dependent. It is machine dependent, fast to run. But it is not easy to understand.</p>
<p><b>High Level language</b> is not specific to one machine i.e. machine independent. It is easy to understand.</p>
