		<h1 class="firstheading"><a href="http://www.codingunit.com/c-tutorial-structures-unions-typedef" rel="bookmark" title="C Tutorial &#8211; structures, unions, typedef">
		C Tutorial &#8211; structures, unions, typedef</a></h1>

			<div class="entry">
				<div id="watermark">
					<p>In the C language structures are used to group together different types of variables under the same name. For example you could create a structure &#8220;telephone&#8221;: which is made up of a string (that is used to hold the name of the person) and an integer (that is used to hold the telephone number).<br />
Take a look at the example:<span id="more-211"></span></p>
<div id="cl">
<pre><code>
	struct telephone
	{
		char *name;
		int number;
	};

</code></pre>
</div>
<p><strong>Note:</strong> the ; behind the last curly bracket.</p>
<p>With the declaration of the structure you have created a new type, called telephone. Before you can use the type telephone you have to create a variable of the type telephone. Take a look at the following example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	struct telephone
	{
		char *name;
		int number;
	};

	int main()
	{
		struct telephone index;

		return 0;
	}

</code></pre>
</div>
<p><strong>Note:</strong> index is now a variable of the type telephone.</p>
<p>To access the members of the structure telephone, you must use a dot between the structure name and the variable name(variables:name or number.) Take a look at the next example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	struct telephone
	{
		char *name;
		int number;
	};

	int main()
	{
		struct telephone index;

		index.name = "Jane Doe";
		index.number = 12345;
		printf("Name: %s\n", index.name);
		printf("Telephone number: %d\n", index.number);

		return 0;
	}

</code></pre>
</div>
<h2>Type definitions and structures</h2>
<p>Type definitions make it possible to create your own variable types. In the following example we will create a type definition called &#8220;intpointer&#8221; (a pointer to an integer):</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	typedef int *int_ptr;

	int main()
	{
		int_ptr myvar;
		return 0;
	}

</code></pre>
</div>
<p>It is also possible to use type definitions with structures. The name of the type definition of a structure is usually in uppercase letters. Take a look at the example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	typedef struct telephone
	{
		char *name;
		int number;
	}TELEPHONE;

	int main()
	{
		TELEPHONE index;

		index.name = "Jane Doe";
		index.number = 12345;
		printf("Name: %s\n", index.name);
		printf("Telephone number: %d\n", index.number);

		return 0;
	}

</code></pre>
</div>
<p><strong>Note:</strong> The word struct is not needed before TELEPHONE index;</p>
<h2>Pointer to structures</h2>
<p>If you want a pointer to a structure you have to use the -&gt; (infix operator) instead of a dot.<br />
Take a look at the following example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	typedef struct telephone
	{
		char *name;
		int number;
	}TELEPHONE;

	int main()
	{
		TELEPHONE index;
		TELEPHONE *ptr_myindex;

		ptr_myindex = &amp;index;

		ptr_myindex-&gt;name = "Jane Doe";
		ptr_myindex-&gt;number = 12345;
		printf("Name: %s\n", ptr_myindex-&gt;name);
		printf("Telephone number: %d\n", ptr_myindex-&gt;number);

		return 0;
	}

</code></pre>
</div>
<p><strong>Note:</strong> The -&gt; (infix operator) is also used in the printf statement.</p>
<h2>Unions</h2>
<p>A union is like a structure in which <strong>all members</strong> are stored at the <strong>same</strong> address. Members of a union can only be accessed one at a time. The union data type was invented to prevent memory fragmentation. The union data type prevents fragmentation by creating a standard size for certain data. Just like with structures, the members of unions can be accessed with the . and -&gt; operators. Take a look at the example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	typedef union myunion
	{
		double PI;
		int B;
	}MYUNION;

	int main()
	{
		MYUNION numbers;
		numbers.PI = 3.14;
		numbers.B = 50;

	return 0;
	}

</code></pre>
</div>
<p>That&#8217;s all for this tutorial.</p>
