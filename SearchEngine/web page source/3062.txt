<h1 class="postitle">C &#8211; Union</h1><hr />
<h4><a href="http://fresh2refresh.com/c/c-object-exe-file-creation/"></a>  <a title="C  Typedef" href="http://fresh2refresh.com/c/c-typedef/">Prev</a>                                              <a title="C  Preprocessor directives" href="http://fresh2refresh.com/c/c-preprocessor-directives/">Next</a></h4>
<p style="text-align: justify;">   C Union is also like structure, i.e. collection of different data types which are grouped together. Each element in a union is called member.</p>
<ul>
<li style="text-align: justify;">Union and structure in C are same in concepts, except allocating memory for their members.</li>
<li style="text-align: justify;">Structure allocates storage space for all its members separately.</li>
<li style="text-align: justify;">Whereas, Union allocates one common storage space for all its members</li>
</ul>
<ul style="text-align: justify;">
<li>We can access only one member of union at a time. We can&#8217;t access all member values at the same time in union. But, structure can access all member values at the same time. This is because,Union allocates one common storage space for all its members. Wher<strong>e</strong>asStructure allocates storage space for all its members separately.</li>
<li>Many union variables can be created in a program and memory will be allocated for each union variable separately.</li>
</ul>
<ul>
<li style="text-align: justify;">Below table will help you how to form a C union, declare a union, initializing and accessing the members of the union.</li>
</ul>

<h4><strong style="font-size: 1em;">Example program for C union:</strong></h4>
</div>
<div>
<pre class="lang:c decode:true">#include &lt;stdio.h&gt;
#include &lt;string.h&gt;

union student 
{
            char name[20];
            char subject[20];
            float percentage;
};

int main() 
{
    union student record1;
    union student record2;

    // assigning values to record1 union variable
       strcpy(record1.name, "Raju");
       strcpy(record1.subject, "Maths");
       record1.percentage = 86.50;

       printf("Union record1 values example\n");
       printf(" Name       : %s \n", record1.name);
       printf(" Subject    : %s \n", record1.subject);
       printf(" Percentage : %f \n\n", record1.percentage);

    // assigning values to record2 union variable
       printf("Union record2 values example\n");
       strcpy(record2.name, "Mani");
       printf(" Name       : %s \n", record2.name);

       strcpy(record2.subject, "Physics");
       printf(" Subject    : %s \n", record2.subject);

       record2.percentage = 99.50;
       printf(" Percentage : %f \n", record2.percentage);
       return 0;
}</pre>

<div style="text-align: left;">Union record1 values example<br />
Name :<br />

Union record2 values example<br />
Name : Mani<br />
Subject : Physics<br />

</div>

<h4><strong>Explanation for above C union program:</strong></h4>
<p style="text-align: justify;"><strong>  </strong>There are 2 union variables declared in this program to understand the difference in accessing values of union members.</p>
<p><strong>Record1 union variable:</strong></p>
<ul>
<li style="text-align: justify;">Raju is assigned to union member record1.name . The memory location name is record1.name and the value stored in this location is &#8220;Raju&#8221;.</li>
<li style="text-align: justify;">Then, Maths is assigned to union member record1.subject. Now, memory location name is changed to record1.subject with the value Maths (Union can hold only one member at a time).</li>
<li style="text-align: justify;">Then, 86.50 is assigned to union member record1.percentage. Now, memory location name ischanged to record1.percentage with value 86.50.</li>
<li style="text-align: justify;">Like this, name and value of union member is replaced every time on the common storage space.</li>
<li style="text-align: justify;">So, we can always access only one union member for which value is assigned at last. We cant access other member values.</li>
<li style="text-align: justify;">So, only &#8220;record1.percentage&#8221; value is displayed in output. &#8220;record1.name&#8221; and &#8220;record1.percentage&#8221; are empty.</li>
</ul>
<p><strong>Record2 union variable:</strong></p>
<ul>
<li style="text-align: justify;">If we want to access all member values using union, we have to access the member before assigning values to other members as shown in record2 union variable in this program.</li>
<li style="text-align: justify;">Each union members are accessed in record2 example immediately after assigning values to them.</li>
<li style="text-align: justify;">If we don&#8217;t access them before assigning values to other member, member name and value will be over written by other member as all members are using same memory.</li>
<li style="text-align: justify;">We can&#8217;t access all members in union at same time but structure can do that.</li>
</ul>
<h4><strong style="font-size: 1em;">Example program &#8211; Another way of declaring C union:</strong></h4>
<p style="text-align: justify;"><strong> </strong>In this program, union variable &#8220;record&#8221; is declared while declaring union itself as shown in the below program.</p>
<pre class="lang:c decode:true">#include &lt;stdio.h&gt;
#include &lt;string.h&gt;

union student 
{
            char name[20];
            char subject[20];
            float percentage;
}record;

int main() 
{

            strcpy(record.name, "Raju");
            strcpy(record.subject, "Maths");
            record.percentage = 86.50;

            printf(" Name       : %s \n", record.name);
            printf(" Subject    : %s \n", record.subject);
            printf(" Percentage : %f \n", record.percentage);
            return 0;
}</pre>

<h4 style="text-align: left;"><strong>Note:</strong></h4>
<p style="text-align: justify;"><strong>  </strong>We can access only one member of union at a time. We can&#8217;t access all member values at the same time in union. But, structure can access all member values at the same time. This is because,Union allocates one common storage space for all its members. Wher<strong>e </strong>asStructure allocates storage space for all its members separately.</p>
<h4 style="text-align: left;"><strong>Difference between structure and union in C:</strong></h4>
<table>
<tbody>
<tr>
<td style="border: 1px solid #000000; text-align: center;"><strong>S.no</strong></td>
<td style="border: 1px solid #000000; text-align: center;">
<div><strong>C Structure</strong></div>
</td>
<td style="border: 1px solid #000000; text-align: center;">
<div><strong>C Union</strong></div>
</td>
</tr>
<tr>
<td style="border: 1px solid #000000; text-align: center;">1</td>
<td style="border: 1px solid #000000;">
<p style="text-align: left;">Structure allocates storage space for all its members separately.</p>
</td>
<td style="border: 1px solid #000000;"><span style="line-height: 19px;">Union allocates one common storage space for all its members.</span><br />
<span style="line-height: 19px;">Union finds that which of its member needs high storage space over other members and allocates that much space</span></td>
</tr>
<tr>
<td style="border: 1px solid #000000; text-align: center;">2</td>
<td style="border: 1px solid #000000;"><span style="line-height: 19px;">Structure occupies higher memory space</span><span style="line-height: 19px;">.</span></td>
<td style="border: 1px solid #000000;"><span style="line-height: 19px;">Union occupies lower memory space over structure.</span></td>
</tr>
<tr>
<td style="border: 1px solid #000000; text-align: center;">3</td>
<td style="border: 1px solid #000000;">We can access all members of structure at a time.</td>
<td style="border: 1px solid #000000;">We can access only one member of union at a time.</td>
</tr>
<tr>
<td style="border: 1px solid #000000; text-align: center;">4</td>
<td style="border: 1px solid #000000;">Structure example:<br />
struct student<br />
{<br />
int mark;<br />
char name[6];<br />
double average;<br />
};</td>
<td style="border: 1px solid #000000;">Union example:<br />
union student<br />
{<br />
int mark;<br />
char name[6];<br />
double average;<br />
};</td>
</tr>
<tr>
<td style="border: 1px solid #000000; text-align: center;">5</td>
<td style="border: 1px solid #000000;"><span style="line-height: 19px;">For above structure, memory allocation will be like below.</span><br />
<span style="line-height: 19px;">int mark &#8211; 2B</span><br />
<span style="line-height: 19px;">char name[6] &#8211; 6B</span><br />
<span style="line-height: 19px;">double average &#8211; 8B</span><br />
<span style="line-height: 19px;">Total memory allocation = 2+6+8 = 16 Bytes</span></td>
<td style="border: 1px solid #000000;"><span style="line-height: 19px;">For above union, only 8 bytes of memory will be allocated since double data type will occupy maximum space of memory over other data types.</span><span style="line-height: 19px;"></span><br />
<span style="line-height: 19px;">Total memory allocation = 8 Bytes</span></td>
</tr>
