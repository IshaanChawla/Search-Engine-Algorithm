<div class = leftMargin>
  <p>A rather fun function you can use is <b>str_shuffle( )</b>. What this does 
    is to shuffle all the characters in a string. You can use this to create a 
    quick anagram programme. Try this script:</p>
  <p class="indent_big">&lt;?PHP</p>
  <p class="indent_bigger">$full_name = 'anagram';</p>
  <p class="indent_bigger">$full_name = str_shuffle($full_name);</p>
  <p class="indent_bigger">print $full_name;</p>
  <p class="indent_big">?&gt;</p>
  <p>Only three lines long, but the function takes a variable or direct text and 
    shuffles the characters around. </p>
