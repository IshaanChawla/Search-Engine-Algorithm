<a href="http://static.springsource.org/spring-batch/reference/html/domain.html" target="_blank">Spring Source</a></em></p><p><a href="http://static.springsource.org/spring-batch/" Spring Batch provides classes and APIs to read/write resources, transaction management, job processing statistics, job restart and partitioning techniques to process high-volume of data.</p><p>The following Spring Batch tutorials and examples are tested with :</p><ol><li>Spring Core 3.2.2.RELEASE</li><li>Spring Batch 2.2.0.RELEASE</li></ol><p><em>P.S Spring Batch is part of the Spring Portfolio.</em></p><h2>1.
