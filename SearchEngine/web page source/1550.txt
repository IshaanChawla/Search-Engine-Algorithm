<p><strong>A first look at booleans in if-statements</strong></p>
<p>One of the most common uses for boolean variables is inside <em>if</em> statements.  If statements take the following form:</p>
<pre>
if (condition) statement
</pre>
<p>or</p>
<pre>
if (condition) statement
else statement2;
</pre>
<p>In both forms, <em>condition</em> is evaluated.  If <em>condition</em> evaluates to true (non-zero), then <em>statement</em> is executed.  In the second form, if <em>condition</em> evaluates to false, then <em>statement2</em> is executed instead.</p>
