<div style="color:#000080;font-family:Arial;font-size:10pt">
<p><b>Stack </b>is a subclass of <b>Vector </b>that implements a
standard last-in, first-out stack. <b>Stack </b>only defines the default constructor, which creates an empty stack. <b>Stack </b>includes all the methods defined by <b>Vector</b>, and adds several of its
own.</p>
<p>To put an object on the top of the stack, call <b>push()</b>.
To remove and return the top element, call <b>pop()</b>. An <b>EmptyStackException </b>is
thrown if you call <b>pop( ) </b>when the invoking stack is empty. You can use <b>peek( ) </b>to return,
but not remove, the top object.</p>
<p>The <b>empty() </b>method returns <b>true </b>if nothing is on
the stack. The <b>search() </b>method determines whether an object exists on the stack, and returns
the number of pops that are required to bring it to the top of the stack. Here is an
example that creates a stack, pushes several <b>Integer </b>objects onto it, and then pops
them off again:</p>
<p>
  <code>// Demonstrate the Stack class.
<br>import java.util.*;
<br>class StackDemo {
<br>static void showpush(Stack st, int a) {
<br>st.push(new Integer(a));
<br>System.out.println(&quot;push(&quot; + a + &quot;)&quot;);
<br>System.out.println(&quot;stack: &quot; + st);
<br>}
<br>static void showpop(Stack st) {
<br>System.out.print(&quot;pop -&gt; &quot;);
<br>Integer a = (Integer) st.pop();
<br>System.out.println(a);
<br>System.out.println(&quot;stack: &quot; + st);
<br>}
<br>public static void main(String args[]) {
<br>Stack st = new Stack();
<br>System.out.println(&quot;stack: &quot; + st);
<br>showpush(st, 42);
<br>showpush(st, 66);
<br>showpush(st, 99);
<br>showpop(st);
<br>showpop(st);
<br>showpop(st);
<br>try {
<br>showpop(st);
<br>} catch (EmptyStackException e) {
<br>System.out.println(&quot;empty stack&quot;);
<br>}
<br>}
<br>}</code>
</p>
<p>The following is the output produced by the program; notice how
the exception handler for <b>EmptyStackException </b>is caught so that you can
gracefully handle a stack underflow:</p>
<p>
<code>stack: [ ]
<br>push(42)
<br>stack: [42]
<br>push(66)
<br>stack: [42, 66]
<br>push(99)
<br>stack: [42, 66, 99]
<br>pop -&gt; 99
<br>stack: [42, 66]
<br>pop -&gt; 66
<br>stack: [42]
<br>pop -&gt; 42
<br>stack: [ ]
<br>pop -&gt; empty stack
</code></p>
