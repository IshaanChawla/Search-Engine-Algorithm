<td  id="content" colspan="7"><h1 class="book_title">More Servlets and JavaServer Pages</h1>
<p class="byline">by Marty Hall</p>
<div class="book_image" style="margin: 0px 0px 7px 15px;"><img src="images/MSAJSP-Book.jpg" alt="MSAJSP" width="141" height="192" /></div>
<ul style="margin: 5px 0px 0px 210px;">
  
  <li><b><font color="red">This book is now freely available online in PDF.</font></b><br/>Please see
  <a href="http://pdf.moreservlets.com/">http://pdf.moreservlets.com/</a>.
    There is also a free online servlet and JSP tutorial with complete source code that has been updated to the latest
    servlet and JSP versions, uses Eclipse, and has exercises and exercise solutions. 
    See <a href="http://courses.coreservlets.com/Course-Materials/csajsp2.html">http://courses.coreservlets.com/Course-Materials/csajsp2.html</a>.<br/><br/>
  </li>
  
  <li><b><a href="http://archive.moreservlets.com/">
    Source code archive.</a></b><br />
    Code for every example in book; on-line links for every
    URL cited in book.</li>
  
  <li><b><a href="http://www.coreservlets.com/Apache-Tomcat-Tutorial/">Using Apache Tomcat.</a></b><br />
    Configuration and usage guide for developing and testing servlets and JSP pages
    under Apache Tomcat.</li>
  
  <li><b><a href="Table-of-Contents.html">Table of contents</a></b></li>
  
  <li><b><a href="http://www.coreservlets.com/">
    Core Servlets &amp; JSP, 2nd Edition.</a></b><br />
    The companion to <i>More Servlets &amp; JSP</i>.<br /> Visit
    <a href="http://volume1.coreservlets.com/">the home page</a> or check out
    <a href="http://courses.coreservlets.com/Course-Materials/csajsp2.html">the free 
      training course materials</a>.</li>
</ul>
<p style="clear: both;">
  The Java 2 Platform has become the technology of choice for developing
  processional e-commerce applications, dynamic Web pages, and Web-enabled
  applications and services. Servlet and JSP technology is the foundation
  of this platform: it provides the link between Web clients
  and server-side applications. But the field has been evolving
  rapidly, and few developers have been able to keep up. In this
  companion to the worldwide bestseller
  <a href="http://volume1.coreservlets.com/">Core Servlets and
    JavaServer Pages</a>, Marty Hall shows you how to apply
  recent advances in servlet and JSP technology. The book provides
  everything you need to know to leverage the latest servlet 2.3
  and JSP 1.2 standards: real-world insight, advanced techniques, 
  industrial-strength code, and hands on coverage of three 
  top servers: Apache Tomcat, Macromedia JRun, and New Atlanta
  ServletExec.
</p>
<p>
  <b>Part I gives a thorough introduction to programming with
    servlet and JSP technology.</b>
  <br />It shows you how to configure your server, read form data 
  and HTTP headers, handle cookies, track
  sessions, apply JSP scripting elements, use JavaBeans 
  components, develop JSP tag libraries, and apply
  the MVC architecture. 
</p>
<p>
  <b>Part II provides exhaustive details on Web application development and
    deployment.</b>
  <br />It explains how to register Web applications, 
  how to organize them, how to deploy them in
  WAR files, how to deal with relative URLs, and 
  how to share data among Web applications. It also gives
  details on <i>every</i> element in version 2.3 of the 
  deployment descriptor (<i>web.xml</i>).
</p>
<p>
  <b>Part III describes Web application security in detail.</b>
  <br />It explains two general strategies for securing your applications: declarative
  security and programmatic security. Within each of these 
  strategies, it shows you how to use form-based
  or BASIC authentication and how to protect your network traffic with SSL.
</p>
<p>
  <b>Part IV covers two features
    introduced with servlets 2.3: filters and life-cycle events.</b>
  <br />It explains how to use filters to debug, modify,
  and optimize the output of previously existing servlets 
  and JSP pages. It also shows you how to use event
  listeners to respond to major events in the server life cycle.
</p><p>
  <b>Part V looks at new tag library developments.</b>
  <br />It shows you how to improve your own tag libraries by making use 
  of new capabilities of the JSP 1.2
  specification and explains how to streamline your code 
  by using the new standard JSP tag library (JSTL).
</p>
