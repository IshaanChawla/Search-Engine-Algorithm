<li style="text-align: justify;">C data types are defined as the data storage format that a variable can store adata to perform a specific operation.</li>
<li>Data types are used to define a variable before to use in a program.</li>
<li>Size of variable, constant and array are determined by data types.</li>
</ul>

<p>There are four data types in C language. They are,</p>

<h4><strong>1. Basic data types in C:</strong></h4>
<h4><strong>1.1. Integer data type:</strong></h4>
<ul>
<li style="text-align: justify;">Integer data type allows a variable to store numeric values.</li>
<li style="text-align: justify;">&#8220;int&#8221;keyword is used to refer integer data type.</li>
<li style="text-align: justify;">The storage size of int data type is 2 or 4 or 8 byte.</li>
<li style="text-align: justify;">It varies depend upon the processor in the CPU that we use. <strong> </strong>If we are using 16 bit processor, 2 byte (16 bit) of memory will be allocated for int data type.</li>
<li style="text-align: justify;">Like wise,4 byte (32 bit) of memory for 32 bit processor and 8byte (64 bit) of memory for 64 bit processor is allocated for int datatype.</li>
<li style="text-align: justify;">int (2 byte) can store values from -32,768 to +32,767</li>
<li style="text-align: justify;">int (4 byte) can store values from -2,147,483,648 to +2,147,483,647.</li>
<li style="text-align: justify;">If you want to use the integer value that crosses the above limit, you can go for &#8220;long int&#8221; and &#8220;long long int&#8221; for which the limits are very high.</li>
</ul>
<p><strong>Note:</strong></p>
<ul>
<li style="text-align: justify;">We can&#8217;t store decimal values using int data type.</li>
<li style="text-align: justify;">If we useint data type to store decimal values, decimal values will be truncated and we will get only whole number.</li>
<li style="text-align: justify;">In this case, float data type can be used to store decimal values in a variable.</li>
</ul>
<h4><strong>1.2. Character data type:</strong></h4>
<ul>
<li>Character data type allows a variable to store only one character.</li>
<li>Storage size of character data type is 1. We can store only one character using character data type.</li>
<li>&#8220;char&#8221; keyword is used to refer characterdata type.</li>
<li>For example, &#8216;A&#8217; can be stored using char datatype.You can&#8217;t store more than one character using char data type.</li>
<li>Please refer <span style="color: #0000ff;"><a title="C  String" href="http://fresh2refresh.com/c/c-strings/"><span style="color: #0000ff;">C &#8211; Strings</span></a></span> topic to know how to store more than one characters in a variable.</li>
</ul>
<h4><strong>1.3. Floating point data type:</strong></h4>
<p>Floating point data type consists of 2 types. They are,</p>
<ol>
<li>float</li>
<li>double</li>
</ol>
<h4><strong>1. float:</strong></h4>
<ul>
<li>Float data type allows a variable to store decimal values.</li>
<li>Storage size of float data type is 4.This also varies depend upon the processor in the CPU as &#8220;int&#8221; data type.</li>
<li>We can use up-to 6 digits after decimal using float data type.</li>
<li>For example, 10.456789 can be stored in a variable using float data type.</li>
</ul>
<h4><strong>2. double:</strong></h4>
<ul>
<li>Double data type is also same as float data type which allows up-to 10 digits after decimal.</li>
<li>The range for double datatype is from1E37 to 1E+37.</li>
</ul>

<p>sizeof() function is used to find the memory space allocated for each C data types.</p>

<h4 style="text-align: left;"><strong>1.3.2. Modifiers in C:</strong></h4>
<ul>
<li style="text-align: justify;">The amount of memory space to be allocated for a variable is derived by modifiers.</li>
<li style="text-align: justify;">Modifiers are prefixed with basic data types to modify (either increase or decrease) the amount of storage space allocated to a variable.</li>
<li style="text-align: justify;">For example, storage space for int data type is 4 byte for 32 bit processor.We can increase the range by using long int which is 8 byte.We can decrease the range by using short int which is 2 byte.</li>
</ul>

<h4><strong style="font-size: 1em;"><strong>2. Enumeration data type in C:</strong></strong></h4>
<ul>
<li>Enumeration data type consists of named integer constants as a list.</li>
<li>It start with 0 (zero) by default and value is incremented by 1 for the sequential identifiers in the list.</li>
</ul>
<ul>
<li>Enum syntax in C:</li>
</ul>
<p style="padding-left: 30px;"><span style="color: #000000;">enum identifier [optional{ enumerator-list }];</span></p>
<ul>
<li>Enum example in C:<strong></strong></li>
</ul>
<p style="padding-left: 30px;">enum month { Jan, Feb, Mar }; or<br />
/* Jan, Feb and Mar variables will be assigned to 0, 1 and 2 respectively by default */<br />
enum month { Jan = 1, Feb, Mar };<br />
/* Feb and Mar variables will be assigned to 2 and 3 respectively by default */<br />
enum month { Jan = 20, Feb, Mar };<br />
/* Jan is assigned to 20. Feb and Mar variables will be assigned to 21 and 22 respectivelyby default */</p>
<ul>
<li>The above enum functionality can also be implemented by &#8220;#define&#8221; preprocessor directive as given below. Above enum example is same as given below.</li>
</ul>
