    <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even" property="content:encoded"><p>Strings are often needed to be manipulated by programmer according to the need of a problem. All string manipulation can be done manually by the programmer but, this makes programming complex and large. To solve this, the C supports a large number of string handling functions.</p><p>There are numerous functions defined in <code>"string.h"</code> header file. Few commonly used string handling functions are discussed below:</p><table class="protable" style="width: 600px; " summary="Function under &quot;string.h&quot; header file in C programming" border="1" cellpadding="1" cellspacing="1"><thead><tr><th scope="col">Function</th><th scope="col">Work of Function</th></tr></thead><tbody><tr><td><a href="http://www.programiz.com/c-programming/library-function/strlen">strlen()</a></td><td>Calculates the length of string</td></tr><tr class="cprotable"><td><a href="http://www.programiz.com/c-programming/library-function/strcpy">strcpy()</a></td><td>Copies a string to another string</td></tr><tr><td><a href="http://www.programiz.com/c-programming/library-function/strcat">strcat()</a></td><td>Concatenates(joins) two strings</td></tr><tr class="cprotable"><td><a href="http://www.programiz.com/c-programming/library-function/strcmp">strcmp()</a></td><td>Compares two string</td></tr><tr><td><a href="http://www.programiz.com/c-programming/library-function/strlwr">strlwr()</a></td><td>Converts string to lowercase</td></tr><tr class="cprotable"><td><a href="http://www.programiz.com/c-programming/library-function/strupr">strupr()</a></td><td>Converts string to uppercase</td></tr></tbody></table><p>Strings handling functions are defined under <code>"string.h"</code> header file, i.e, you have to include the code below to run string handling functions.</p>
<pre>#include &lt;string.h&gt;</pre>
<h3>gets() and puts()</h3><p>Functions gets() and puts() are two string functions to take string input from user and display string respectively as mentioned in previous chapter.</p>
<pre><code>#include&lt;stdio.h&gt;
int main(){
    char name[30];
    printf("Enter name: ");
    gets(name);     //Function to read string from user.
    printf("Name: ");
    puts(name);    //Function to display string.
    return 0;
}</code></pre>
<p>Though, gets() and <code>puts()</code> function handle string, both these functions are defined in <code>"stdio.h"</code> header file.</p></div></div></div>
