    <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even"><p>C programming language: To easy learn c you must start making programs in it. As you may already know that to develop programs you need a text editor and  a compiler to translate source program into machine code which can be executed directly on machine. Dev C++ IDE is a good choice, so if you are not having it installed on your computer then download <a href="http://sourceforge.net/projects/orwelldevcpp/files/latest/download" rel="nofollow" target="_blank" title="Download Software: Dev C++ compiler">Download Dev c++ compiler</a>. </p>
<h2>C programming code</h2>
<p>Now you have the tool to create programs, if you wish to look at some example codes then see <a href="http://www.programmingsimplified.com/c-program-examples" title="c programming examples">c programming examples</a>. Below I explain you to how to compile and run your programs. You may be using different compilers or operating system. I will explain you how to use Dev C++, GCC and turbo c compiler.</p>
<p>Dev C++<br />
1. Create new file (Ctrl + N).<br />
2. Write the code<br />
3. Press Ctrl+F9 to compile.<br />
4. Press Ctrl+F10 to run.</p>
<p>Compiling and running c programs in GCC compiler<br />
1. Save the code in a file say "hello.c" (quotes for clarity).<br />
2. To compile open terminal and type "gcc hello.c".<br />
3. To execute type "./a.out".</p>
<p>You can also specify the output file name as follows:<br />
gcc hello.c -o hello.out<br />
hello.out is name of output file.</p>
<p>Turbo C compiler:-<br />
1. Write your code in a file and save it.<br />
2. Compile by pressing Alt + F9.<br />
3. Execute by pressing Ctrl + F9<br />
4. For output of this program, press (Alt + F5).</p>
<h2>C programming basics</h2>
<p>Computer programming means giving instruction to the computer and to interact with the computer we need a language to communicate with it, there are many languages such as C, C++, Java, Python each having their own features. Let's discuss first why we need programming suppose you are given ten numbers and asked to arrange them in ascending order then you can do it easily, but what if the numbers are say ten thousand it will be a difficult task and may take a long time and result may also contain errors, to simplify we can write a program which does this task and if the algorithm is correct then we will get result accurately and quickly, this is where programming is helpful.</p>
