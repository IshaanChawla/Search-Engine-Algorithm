    <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even" property="content:encoded"><p>In C programming, one of the frequently arising problem is to handle similar types of data. For example: If the user want to store marks of 100 students. This can be done by creating 100 variable individually but, this process is rather tedious and impracticable. These type of problem can be handled in C programming using arrays.</p>
<p>An array is a sequence of data item of homogeneous value(same type).</p>
<p>Arrays are of two types:</p>
<ol><li>One-dimensional arrays</li><li><a href="http://www.programiz.com/c-programming/c-multi-dimensional-arrays" title="Multi-dimensional Arrays In C Programming">Multidimensional arrays</a>( will be discussed in next chapter )</li></ol>
<h3>Declaration of one-dimensional array</h3>

<pre>
data_type array_name[array_size];

<strong>For example</strong>:
int age[5];</pre>

<p>Here, the name of array is <var>age</var>. The size of array is 5,i.e., there are 5 items(elements) of array <var>age</var>. All element in an array&nbsp;are of the same type (int, in this case).</p>
<h3>Array elements</h3>
<p>Size of array defines the number of elements in an array. Each element of array can be accessed and used by user according to the need of program. For example:</p>

<pre>
int age[5];
</pre>

<p><img alt="Elements of arrays in C." class="figure" src="../sites/tutorial2program/files/C_arrays_elements.jpg" style="height:71px; width:295px" title="Array elements" /></p>
<p>Note that, the first element is numbered 0 and so &nbsp;on.</p>
<p>Here, the size of array <var>age</var> is 5 times the size of int because there are 5 elements.</p>
<p>Suppose, the starting address of <code>age[0]</code> is 2120d and the size of int be 4 bytes. Then, the next address (address of <code>a[1]</code>) will be 2124d, address of <code>a[2]</code> will be 2128d and so on.</p>
<h3>Initialization of one-dimensional array:</h3>
<p>Arrays can be initialized at declaration time in &nbsp;this source code as:</p>

<pre>
int age[5]={2,4,34,3,4};
</pre>

<p>It is not necessary to define the size of arrays during initialization.</p>

<pre>
int age[]={2,4,34,3,4};
</pre>

<p>In this case, the compiler determines the size of array by calculating the number of elements of an array.</p>
<p><img alt="Initialization of one-dimensional arrays in C programming" class="figure" src="../sites/tutorial2program/files/C_Initialization_of_arrays.jpg" style="height:74px; width:295px" title="Initialization of array in memory" /></p>
<h3>Accessing array elements</h3>
<p>In C programming, arrays can be accessed and treated like variables in C.</p>
<p>For example:</p>

<pre>
scanf(&quot;%d&quot;,&amp;age[2]);
/* statement to insert value in the third element of array age[]. */

scanf(&quot;%d&quot;,&amp;age[i]);
/* Statement to insert value in (i+1)th element of array age[]. */
/* Because, the first element of array is age[0], second is age[1], ith is age[i-1] and (i+1)th is age[i]. */

printf(&quot;%d&quot;,age[0]);
/* statement to print first element of an array. */

printf(&quot;%d&quot;,age[i]);
/* statement to print (i+1)th element of an array. */
</pre>

<h3>Example of array in C programming</h3>

<pre>
<code>
/* C program to find the sum marks of n students using arrays */

#include &lt;stdio.h&gt;
int main(){
     int marks[10],i,n,sum=0;
     printf(&quot;Enter number of students: &quot;);
     scanf(&quot;%d&quot;,&amp;n);
     for(i=0;i&lt;n;++i){
          printf(&quot;Enter marks of student%d: &quot;,i+1);
          scanf(&quot;%d&quot;,&amp;marks[i]);
          sum+=marks[i];
     }
     printf(&quot;Sum= %d&quot;,sum);
return 0;
}
</code>
</pre>

<p><strong>Output</strong></p>

<pre>
<samp>Enter number of students: 3
Enter marks of student1: 12
Enter marks of student2: 31
Enter marks of student3: 2
sum=45
</samp></pre>

<h3>Important thing to remember in C arrays</h3>
<p>Suppose, you declared the array of 10 students. For example: <code>arr[10]</code>. You can use array members from <code>arr[0]</code> to <code>arr[9]</code>. But, what if you want to use element <code>arr[10]</code>, <code>arr[13]</code> etc. Compiler may not show error using these elements but, may cause fatal error during program execution.</p></div></div></div>
