                  <div>
                     <h2 class="title"><a name="jdbc"></a>1.&nbsp;Connection to database with Java
                     </h2>
                  </div>
               </div>
            </div>
            <p>
               The interface for accessing relational databases from Java is
               <em class="wordasword">Java Database Connectivity (JDBC)</em>. Via JDBC you create a connection to the database, issue database
               queries and update as well as receive the results.
               
            </p>
            <p>
               JDBC provides an interface which allows you to perform SQL
               operations independently of the instance of the used database. To use
               JDBC, you require the database specific implementation of the
               JDBC
               driver.
               
            </p>
         </div>
         <div class="section">
            <div class="titlepage">
               <div>
                  <div>
                     <h2 class="title"><a name="overview"></a>2.&nbsp;Introduction to MySQL
                     </h2>
                  </div>
               </div>
            </div>
            <p>
               To learn to install and use MySQL please
               see
               <a class="ulink" href="http://www.vogella.com/tutorials/MySQL/article.html" target="_top">MySQL - Tutorial</a>.
               
            </p>
            <p>
               The following description will assume that you have successfully installed
               MySQL and know how to access MySQL via the command line.
               
            </p>
         </div>

               <div>
                  <div>
                     <h2 class="title"><a name="jdbcdriver"></a>3.&nbsp;MySQL JDBC driver
                     </h2>
                  </div>
               </div>
            </div>
            <p>
               To connect to MySQL from Java, you have to use the JDBC driver
               from
               MySQL. The MySQL JDBC driver is called
               <em class="wordasword">MySQL Connector/J</em>. You find the latest MySQL JDBC driver under the following URL:
               <a class="ulink" href="http://dev.mysql.com/downloads/connector/j" target="_top">http://dev.mysql.com/downloads/connector/j</a>.
               
            </p>
            <p>
               The download contains a
               <code class="filename">JAR</code>
               file which we require later.
               
            </p>

               <div>
                  <div>
                     <h2 class="title"><a name="database"></a>4.&nbsp;Exercise: create example database
                     </h2>
                  </div>
               </div>
            </div>
            <p>
               In this exercise you create a new database, a new user and an
               example
               table. For this
               connect to the MySQL server via the
               <code class="code">mysql</code>
               command line client.
               
            </p>
            <p>
               Create a new database called
               <em class="wordasword">feedback</em>
               and start using it with the following command.
               
            </p>
            <p>
               
            </p><pre class="programlisting">create database feedback;
use feedback; </pre><p>
               
            </p>
            <p>
               Create a user with the following command.
               
            </p>
            <p>
               
            </p><pre class="programlisting">CREATE USER sqluser IDENTIFIED BY 'sqluserpw'; 

grant usage on *.* to sqluser@localhost identified by 'sqluserpw'; 
grant all privileges on feedback.* to sqluser@localhost; </pre><p>
               
            </p>
            <p>
               Now create a sample
               database table with example content via the
               following
               SQL statement.
               
            </p>
            <p>
               
            </p><pre class="programlisting">CREATE TABLE comments (id INT NOT NULL AUTO_INCREMENT, 
    MYUSER VARCHAR(30) NOT NULL,
    EMAIL VARCHAR(30), 
    WEBPAGE VARCHAR(100) NOT NULL, 
    DATUM DATE NOT NULL, 
    SUMMARY VARCHAR(40) NOT NULL,

                  <div>
                     <h2 class="title"><a name="javaconnection"></a>5.&nbsp;Java JDBC
                     </h2>
                  </div>
               </div>
            </div>
            <p>
               Create a Java project and a package called
               <em class="wordasword">de.vogella.mysql.first</em>.
               
            </p>
            <p>
               Create a
               <code class="filename">lib</code>
               folder and copy the JDBC driver into this folder.
               Add
               the JDBC driver to
               your classpath. See
               <a class="ulink" href="http://www.vogella.com/tutorials/Eclipse/article.html#classpath" target="_top">Adding jars to the classpath</a>
               for details.
               
            </p>
            <p>
               Create the following class to connect to the
               MySQL
               database and
               perform queries, inserts and deletes. It also prints the metadata
               (table name, column names) of a query result.
               
            </p>
            <p>
               
            </p><pre class="programlisting"><span class="hl-keyword">package</span> de.vogella.mysql.first;

<span class="hl-keyword">import</span> java.sql.Connection;
<span class="hl-keyword">import</span> java.sql.DriverManager;
<span class="hl-keyword">import</span> java.sql.PreparedStatement;
<span class="hl-keyword">import</span> java.sql.ResultSet;
<span class="hl-keyword">import</span> java.sql.SQLException;
<span class="hl-keyword">import</span> java.sql.Statement;
<span class="hl-keyword">import</span> java.util.Date;

<span class="hl-keyword">public</span> <span class="hl-keyword">class</span> MySQLAccess {
  <span class="hl-keyword">private</span> Connection connect = null;
  <span class="hl-keyword">private</span> Statement statement = null;
  <span class="hl-keyword">private</span> PreparedStatement preparedStatement = null;
  <span class="hl-keyword">private</span> ResultSet resultSet = null;

  <span class="hl-keyword">public</span> <span class="hl-keyword">void</span> readDataBase() <span class="hl-keyword">throws</span> Exception {
    <span class="hl-keyword">try</span> {
      <em class="hl-comment" style="color: #080">// This will load the MySQL driver, each DB has its own driver</em>
      Class.forName(<span class="hl-string" style="color:blue">"com.mysql.jdbc.Driver"</span>);
      <em class="hl-comment" style="color: #080">// Setup the connection with the DB</em>
      connect = DriverManager
          .getConnection(<span class="hl-string" style="color:blue">"jdbc:mysql://localhost/feedback?"</span>
              + <span class="hl-string" style="color:blue">"user=sqluser&amp;password=sqluserpw"</span>);

      <em class="hl-comment" style="color: #080">// Statements allow to issue SQL queries to the database</em>
      statement = connect.createStatement();
      <em class="hl-comment" style="color: #080">// Result set get the result of the SQL query</em>
      resultSet = statement
          .executeQuery(<span class="hl-string" style="color:blue">"select * from feedback.comments"</span>);
      writeResultSet(resultSet);

      <em class="hl-comment" style="color: #080">// PreparedStatements can use variables and are more efficient</em>
      preparedStatement = connect
          .prepareStatement(<span class="hl-string" style="color:blue">"insert into  feedback.comments values (default, ?, ?, ?, ? , ?, ?)"</span>);
      <em class="hl-comment" style="color: #080">// "myuser, webpage, datum, summery, COMMENTS from feedback.comments");</em>
      <em class="hl-comment" style="color: #080">// Parameters start with 1</em>
      preparedStatement.setString(<span class="hl-number">1</span>, <span class="hl-string" style="color:blue">"Test"</span>);
      preparedStatement.setString(<span class="hl-number">2</span>, <span class="hl-string" style="color:blue">"TestEmail"</span>);
      preparedStatement.setString(<span class="hl-number">3</span>, <span class="hl-string" style="color:blue">"TestWebpage"</span>);
      preparedStatement.setDate(<span class="hl-number">4</span>, <span class="hl-keyword">new</span> java.sql.Date(<span class="hl-number">2009</span>, <span class="hl-number">12</span>, <span class="hl-number">11</span>));
      preparedStatement.setString(<span class="hl-number">5</span>, <span class="hl-string" style="color:blue">"TestSummary"</span>);
      preparedStatement.setString(<span class="hl-number">6</span>, <span class="hl-string" style="color:blue">"TestComment"</span>);
      preparedStatement.executeUpdate();

      preparedStatement = connect
          .prepareStatement(<span class="hl-string" style="color:blue">"SELECT myuser, webpage, datum, summery, COMMENTS from feedback.comments"</span>);
      resultSet = preparedStatement.executeQuery();
      writeResultSet(resultSet);

      <em class="hl-comment" style="color: #080">// Remove again the insert comment</em>
      preparedStatement = connect
      .prepareStatement(<span class="hl-string" style="color:blue">"delete from feedback.comments where myuser= ? ; "</span>);
      preparedStatement.setString(<span class="hl-number">1</span>, <span class="hl-string" style="color:blue">"Test"</span>);
      preparedStatement.executeUpdate();
      
      resultSet = statement
      .executeQuery(<span class="hl-string" style="color:blue">"select * from feedback.comments"</span>);
      writeMetaData(resultSet);
      
    } <span class="hl-keyword">catch</span> (Exception e) {
      <span class="hl-keyword">throw</span> e;
    } <span class="hl-keyword">finally</span> {
      close();
    }

  }

  <span class="hl-keyword">private</span> <span class="hl-keyword">void</span> writeMetaData(ResultSet resultSet) <span class="hl-keyword">throws</span> SQLException {
    <em class="hl-comment" style="color: #080">//   Now get some metadata from the database</em>
    <em class="hl-comment" style="color: #080">// Result set get the result of the SQL query</em>
    
    System.out.println(<span class="hl-string" style="color:blue">"The columns in the table are: "</span>);
    
    System.out.println(<span class="hl-string" style="color:blue">"Table: "</span> + resultSet.getMetaData().getTableName(<span class="hl-number">1</span>));
    <span class="hl-keyword">for</span>  (<span class="hl-keyword">int</span> i = <span class="hl-number">1</span>; i&lt;= resultSet.getMetaData().getColumnCount(); i++){
      System.out.println(<span class="hl-string" style="color:blue">"Column "</span> +i  + <span class="hl-string" style="color:blue">" "</span>+ resultSet.getMetaData().getColumnName(i));
    }
  }

  <span class="hl-keyword">private</span> <span class="hl-keyword">void</span> writeResultSet(ResultSet resultSet) <span class="hl-keyword">throws</span> SQLException {
    <em class="hl-comment" style="color: #080">// ResultSet is initially before the first data set</em>
    <span class="hl-keyword">while</span> (resultSet.next()) {
      <em class="hl-comment" style="color: #080">// It is possible to get the columns via name</em>
      <em class="hl-comment" style="color: #080">// also possible to get the columns via the column number</em>
      <em class="hl-comment" style="color: #080">// which starts at 1</em>
      <em class="hl-comment" style="color: #080">// e.g. resultSet.getSTring(2);</em>
      String user = resultSet.getString(<span class="hl-string" style="color:blue">"myuser"</span>);
      String website = resultSet.getString(<span class="hl-string" style="color:blue">"webpage"</span>);
      String summery = resultSet.getString(<span class="hl-string" style="color:blue">"summery"</span>);
      Date date = resultSet.getDate(<span class="hl-string" style="color:blue">"datum"</span>);
      String comment = resultSet.getString(<span class="hl-string" style="color:blue">"comments"</span>);
      System.out.println(<span class="hl-string" style="color:blue">"User: "</span> + user);
      System.out.println(<span class="hl-string" style="color:blue">"Website: "</span> + website);
      System.out.println(<span class="hl-string" style="color:blue">"Summery: "</span> + summery);
      System.out.println(<span class="hl-string" style="color:blue">"Date: "</span> + date);
      System.out.println(<span class="hl-string" style="color:blue">"Comment: "</span> + comment);
    }
  }

  <em class="hl-comment" style="color: #080">// You need to close the resultSet</em>
  <span class="hl-keyword">private</span> <span class="hl-keyword">void</span> close() {
    <span class="hl-keyword">try</span> {
      <span class="hl-keyword">if</span> (resultSet != null) {
        resultSet.close();
      }

      <span class="hl-keyword">if</span> (statement != null) {
        statement.close();
      }

      <span class="hl-keyword">if</span> (connect != null) {
        connect.close();
      }
    } <span class="hl-keyword">catch</span> (Exception e) {

    }
  }

} </pre><p>
