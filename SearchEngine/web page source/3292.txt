<P>Instead of detailing all the possible string functions you can use, we'll just 
  give you a brief list. (There's loads of them!) There's an example of how to 
  use each string function, if you click on the links below. Just dip in to them 
  as and when needed.</P>
<P class="indent_big_no_bold"><a href="php7p8s1.html">chr( )</a> Convert an ASCII value to a its equivalent character</P>
<P class="indent_big_no_bold"><a href="php7p8s2.html">ord( )</a> Find out what the ASCII value of a character 
  is</P>
<P class="indent_big_no_bold"><a href="php7p8s3.html">echo( )</a> An alternative to the print statement</P>
<P class="indent_big_no_bold"><a href="php7p8s4.html">similar_text( )</a> Tells you how similar two strings 
  of text are</P>
<P class="indent_big_no_bold"><a href="php7p8s5.html">str_repeat( )</a> Repeat a character a specified number 
  of times</P>
<P class="indent_big_no_bold"><a href="php7p8s6.html">str_replace( )</a> Replace one string with another</P>
<P class="indent_big_no_bold"><a href="php7p8s7.html">str_word_count( )</a> Tells you how many words a string 
  has</P>
<P class="indent_big_no_bold"><a href="php7p8s8.html">strlen( )</a> Gets the length of a string</P>
<P class="indent_big_no_bold"><a href="php7p8s9.html">substr( )</a> Grab a number of character from a string</P>
