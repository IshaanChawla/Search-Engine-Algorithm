<h1>VBScript Tutorial</h1>
<p>VBScript stands for Visual Basic Script, a scripting language developed by Microsoft to be
used with Microsoft products, mainly Internet Explorer.  It has gone through many changes over
the years and is now mainly used as the <a HREF="http://www.tizag.com/aspTutorial">default scripting language of ASP</a>.
</p>

<h1>VBScript: Client Side Scripting</h1>
<p>VBScript was created to allow web page developers the ability to create dynamic web pages
for their viewers who used Internet Explorer.  With HTML, not a lot can be done to make
a web page interactive, but VBScript unlocked many tools like: the ability to print the current date and time, 
access to the web servers file system, and allow advanced web programmers to develop
web applications.</p>

<h1>VBScript: Why Learn and Use It?</h1>
<p>VBScript is a prerequisite for ASP developers and should be learned thoroughly before
attempting any sophisticated ASP programming.  Programmers who have used Visual Basic in the past
will feel more at home when designing their dynamic web pages with VBScript, but it should be known
that only visitor's using Internet Explorer will be able to access your code, while other browsers will
not be able to process the VBScript code.</p>

<h1>Tizag's VBScript Learning: Do's &amp; Don'ts</h1>
<b>DO Learn VBScript</b>
<ul>
	<li>If you have a solid understanding of HTML</li>
	<li>If you want to program successfully in ASP</li>
	<li>If you already know Visual Basic and would like to experiment with VBScript</li>
	<li>If you are a hobbyist that enjoys knowing many programming </li>
</ul>
<b>DON'T Learn VBScript</b>
<ul>
	<li>If you are developing a web page for the general public</li>
	<li>If your site has visitors who use a browser other that Internet Explorer</li>
	<li>If prefer to avoid dying languages. If you want something more supported, you should <a HREF="http://www.tizag.com/javascriptT/">learn Javascript</a></li>
</ul>

<p>If you still want to learn VBScript then let's get started.</p>
