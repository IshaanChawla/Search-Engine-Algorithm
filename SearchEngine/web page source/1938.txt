<h1 class="content_heading">Sun Certified Web Component Developer (SCWCD)</h1>
	<h1 class="subtitle"> Review and Exam Prep</h1>

	 <p>The training materials on this page are taken from Marty's live SCWCD exam preparation course. This course and several
	   similar J2EE courses have been given by Marty in the US, Canada, Australia, Puerto Rico, Japan, Mexico, and the Philippines.
	   See the <a href="index.html">training materials home page</a> for <a href="struts.html">Jakarta Struts tutorials</a>,
	   <a href="http://www.coreservlets.com/Apache-Tomcat-Tutorial/">Apache Tomcat tutorials</a>, 
	   <a href="http://www.coreservlets.com/Course-Materials/JSF-Tutorial/">
	     JSF tutorials</a>,
	   <a href="java5.html">Java 5 &amp; Java 6 tutorials</a>, and tutorials in <a href="chinese.html">Chinese</a> and 
	   <a href="japanese.html">Japanese</a>.</p>
	 <p>The PDF versions of the tutorials are freely available to anyone for personal use. These versions contain the complete text of 
	   the PowerPoint slides, and are what you want if your goal is to learn or review the technology.
	   As a courtesy, <a href="http://www.coreservlets.com">coreservlets.com</a>
	   also makes the original PowerPoint slides available to faculty teaching university courses that do not compete with
	   Marty's live courses. See <a href="instructor-materials.html">the J2EE instructor materials page</a> for details.</p>
	 <p> To arrange an SCWCD course at <i>your</i> organization based on these materials, contact 
	   <a href="mailto:info@coreservlets.com">info@coreservlets.com</a>. These courses can be customized to use any 
	   combination of the materials on the <a href="index.html">J2EE tutorials site</a>, and new materials can be added for specific
