<h4><strong>Example program for strftime(), asctime()and localtime() in C:</strong></h4>
<ul>
<li>strftime()  - This function is used to modify the actual time format.</li>
<li>asctime()  &#8211; tm structure contents are interpreted by asctime() function as calendar time. This time is converted into string.</li>
<li>localtime() &#8211; This function shares the tm structure that contains date and time informations.</li>
</ul>
<pre class="lang:c decode:true">#include &lt;stdio.h&gt;
#include &lt;time.h&gt;

#define LEN 150
int main ()
{
   char buf[LEN];
   time_t curtime;
   struct tm *loc_time;

   //Getting current time of system
   curtime = time (NULL);

   // Converting current time to local time
   loc_time = localtime (&amp;curtime);

   // Displaying date and time in standard format
   printf("%s", asctime (loc_time));

   strftime (buf, LEN, "Today is %A, %b %d.\n", loc_time);
   fputs (buf, stdout);
   strftime (buf, LEN, "Time is %I:%M %p.\n", loc_time);
   fputs (buf, stdout);

   return 0;
}</pre>
<h4><strong>Output:</strong></h4>
<table style="width: 491px; height: 56px;">
<tbody>
<tr>
<td style="border: 1px solid #000000; text-align: left;">Sat Sep 22 01:15:03 2012<br />
Today is Saturday, Sep 22.<br />
Time is 01:15 AM.</td>
</tr>
</tbody>
</table>
