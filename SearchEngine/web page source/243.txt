    <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even"><p>You can contribute to open source by using open source software. Some of the most useful computer software that you should try:</p>
<p><a href="https://affiliates.mozilla.org/link/banner/9936/3/1" rel="nofollow" title="Firefox web browser">Firefox web browser</a>: Browse the web with this awesome web browser. It's not only open source but also a product of Non Profit Organization(Mozilla Corporation).</p>
<p><a href="http://affiliates.mozilla.org/link/banner/6125" target="_blank" rel="nofollow">Mozilla Thunderbird</a>: Read and send email directly from your desktop.</p>
<p><a href="http://www.rssowl.org/download" target="_blank" rel="nofollow">RSSOwl Newsreader</a>: Get latest news directly without searching or browsing websites.</p>
<p><a href="http://www.videolan.org/vlc/" target="_blank" rel="nofollow">VLC media player</a>: Play your favorite music or videos.</p>
<p><a href="http://www.gimp.org/downloads/" target="_blank" rel="nofollow">GIMP</a>: Edit or create photos, it offers many great features.</p>
<p><a href="http://www.openoffice.org/" target="_blank" rel="nofollow">OpenOffice</a>: Create and edit documents,  presentations, spreadsheets and many more. It is compatible with file formats used by Microsoft Office.</p>
<p><a href="http://www.mirovideoconverter.com/" target="_blank" rel="nofollow">Miro Video Converter</a>: Convert videos to many different formats including Android, iPhone.</p>
<p><a href="http://www.getmiro.com/" target="_blank" rel="nofollow">Miro</a>: Awesome music and video player, allows you to search videos directly from it, browse websites, download torrents, listen to podcast.</p>
<p><a href="http://www.peazip.org/" target="_blank" rel="nofollow">PeaZip</a>: Create and extract archived files.</p>
<p><a href="http://notepad-plus-plus.org/download/" target="_blank" rel="nofollow">Notepad++ text editor</a>: A very useful text editor that allows files to be opened in tabs, advanced search and replace, auto completion and you can customize it by downloading additional plugins.</p>
<p><a href="http://www.ubuntu.com/download" target="_blank" rel="nofollow">Ubuntu Operating System</a>: Linux distribution which is easy to use, configure and learn. If you wish to switch to Linux then it's simple with Ubuntu.</p>
<h2>Play games</h2>
<p>There are many free games that you can download and play. Some of them which i like are:<br />
<a href="http://supertuxkart.sourceforge.net/Downloads#header_win" title="Car racing game" rel="nofollow" target="_blank">Supertuxkart</a>: Solve challenges and compete with others.</p>
