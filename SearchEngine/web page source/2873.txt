		<h1 class="firstheading"><a href="http://www.codingunit.com/c-tutorial-the-if-and-switch-statement" rel="bookmark" title="C Tutorial &#8211; The if and switch statement">
		C Tutorial &#8211; The if and switch statement</a></h1>

			<div class="entry">
				<div id="watermark">
					<p>In this C programming language tutorial we take a look at the &#8220;if statement&#8221; and &#8220;switch statement&#8221;. Both are used to alter the flow of a program if a specified test condition is true.</p>
<h2>Boolean Operators</h2>
<p>Before we can take a look at test conditions we have to know what Boolean operators are. They are called Boolean operators because they give you either true or false when you use them to test a condition. The greater than sign &#8220;&gt;&#8221;<br />
for instance is a Boolean operator. <span id="more-160"></span> In the table below you see all the Boolean operators:</p>

</tr>

<h2>The if statement</h2>
<p>The if statement can be used to test conditions so that we can alter the flow of a program. In other words: if a specific statement is true, execute this instruction. If not true, execute this instruction. So lets take a look at an example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	int main()
	{

		int mynumber;

		scanf("%d",&amp;mynumber);
		if ( mynumber == 10 )
			printf("Is equal\n");
		return 0;
	}

</code></pre>
</div>
<p>In the example above the user can input a number. The number is stored in the variable mynumber.<br />
Now take a look at the &#8220;if statement&#8221;: if the number stored in the variable mynumber is equal to ten, then print &#8220;is equal&#8221; on the screen. Simple, isn&#8217;t it. If the number is not equal to ten, then nothing gets printed.</p>
<p>Now take another look at the &#8220;if statement&#8221;: look at the placement of the semi-colon. As you can see there is no semi-colon behind the &#8220;if statement&#8221;. If there is just one instruction (if the statement is true), you can place it after the<br />
&#8220;if statement&#8221; (with an indentation). Are multiple instructions necessary then you will have to use curly brackets, like so:</p>
<div id="cl">
<pre><code>
	if ( mynumber == 10)
	{
		printf("is equal\n");
		printf("closing program\n");
	}

	</code></pre>
</div>
<p>Now we like to also print something if the &#8220;if statement&#8221; is not equal. We could do this by adding another &#8220;if statement&#8221; but there is an easier / better way. Which is using the so called &#8220;else statement&#8221; with the &#8220;if statement&#8221;.</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	int main()
	{
		int mynumber;
		scanf("%d",&amp;mynumber);

		if ( mynumber == 10 )
		{
			printf("Is equal\n");
			printf("Closing program\n");
		}
		else
		{
			printf("Not equal\n");
			printf("Closing program\n");
		}
		return 0;
	}

</code></pre>
</div>
<p><strong><strong>Note:</strong></strong> Take a look at the placement of the curly brackets and how the indentations are placed.<br />
This is all done to make reading easier and to make less mistakes in large programs.</p>
<h2>Nested if statements</h2>
<p>If you use an &#8220;if statement&#8221; in an &#8220;if statement&#8221; it is called nesting. Nesting &#8220;if statements&#8221; can make a program very complex, but sometimes there is no other way. So use it wisely. Take a look at a nested &#8220;if statement&#8221; example below:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	int main()
	{
		int grade;

		scanf("%d",&amp;grade);
		if ( grade &lt;= 10 )
		{
			printf("YOU DID NOT STUDY.\n");
			printf("YOU FAILED ! \n");
		}
		else
		{
			if ( grade &lt; 60 ) 			{ 				printf("You failed \n"); 				printf("Study harder\n"); 			} 			else 			{ 				if ( grade &gt;= 60 )
					printf("YOU PASSED ! \n");
			}
		}
		return 0;
	}

</code></pre>
</div>
<p><strong><strong>Note:</strong></strong> Again take a look at the placing of the curly brackets and the placing of the indentations.</p>
<p>So let&#8217;s walk through the example above: If the grade is equal or below ten, you did not study. If the grade is above ten, the program will go into the &#8220;else statement&#8221;. In the &#8220;else statement&#8221; there is another &#8220;if statement&#8221; (this is nesting).<br />
This &#8220;if statement&#8221; checks the grade again. If the grade is below sixty, you must study harder. In the &#8220;else statement&#8221; from this &#8220;if statement&#8221; there is another &#8220;if statement&#8221;. It checks whether the grade is above or equal to sixty. If it is, you passed the test.</p>
<h2>Multiple condition testing</h2>
<p>It is possible to test two or more conditions at once in an &#8220;if statement&#8221; with the use of the AND (&amp;&amp;) operator. Example:</p>
<div id="cl">
<pre><code>
	if ( a &gt; 10 &amp;&amp; b &gt; 20 &amp;&amp; c &lt; 10 )
	</code></pre>
</div>
<p>If a is greater then ten and b is greater then twenty and c is smaller then ten, do something. So all three conditions must be true, before something happens.</p>
<p>With the OR ( || ) operator you can test if one of two conditions are true. Example:</p>
<div id="cl">
<pre><code>
	if ( a = 10 || b &lt; 20 )
	</code></pre>
</div>
<p>If a equals ten or b is smaller then twenty then do something. So if a or b is true, something happens.</p>
<h2>The switch statement</h2>
<p>The switch statement is almost the same as an &#8220;if statement&#8221;. The switch statement can have many conditions. You start the switch statement with a condition. If one of the variable equals the condition, the instructions are executed. It is also possible to add a default. If none of the variable equals the condition the default will be executed. See the<br />
example below:</p>
<div id="cl">
<pre><code>
      #include&lt;stdio.h&gt;

	int main()

	{
		char myinput;

		printf("Which option will you choose:\n");
		printf("a) Program 1 \n");
		printf("b) Program 2 \n");
		scanf("%c", &amp;myinput);

		switch (myinput)
		{
			case 'a':
			 		printf("Run program 1\n");
			 		break;
			case 'b':
				{
			 		printf("Run program 2\n");
					printf("Please Wait\n");
					break;
				}
			default:
					printf("Invalid choice\n");
					break;
		}
		return 0;
	}

</code></pre>
</div>
<p><strong><strong>Note:</strong></strong> break is used to exit the switch. See the next tutorial for more details.</p>
<p>That&#8217;s all for this tutorial.</p>
<p>We encourage you to make your own programs or to make changes in the example programs. This is the only way you can learn programming C. Make little programs, the more the better. <strong>Have fun!</strong></p>
