<td  id="content" colspan="7"><h1 class="content_heading">Free Java EE, JSF 2.0, Ajax, Android, jQuery, &amp; GWT Tutorials</h1>
<p>Extensive tutorials, source code, and exercises derived from Marty Hall's
  world-renowned <a href="http://courses.coreservlets.com">live
    J2EE, Ajax, &amp; GWT training courses</a> are now available online. These training materials are based
  on Marty's books <i>Core Servlets and JavaServer Pages</i>,
  <i>More Servlets and JavaServer Pages</i>, and
  <i>Core Web Programming</i>, and various online tutorials
  and conference talks. They have been tested by Marty in live courses
  in the US, Canada, Australia, Puerto Rico, Japan, Mexico, India,
  and the Philippines. <a href="mailto:hall@coreservlets.com">Contact Marty</a>
  to arrange a customized course onsite at <i>your</i> location.</p>

<p>Topics include servlets, JSP, JSF 1.<i>x</i>, JSF 2.0, Struts,
  Ajax (with any of jQuery, Prototype, Scriptaculous, Dojo, or Ext-JS), GWT, Tomcat setup, Java 7 programming, 
  Spring, Hibernate/JPA, and more. 
  The following tutorial series are available (click on the header for more details):</p>

<div class="accordion-div">
  <h1 class="book_part"><b><a name="http://www.coreservlets.com/JSF-Tutorial/">JSF 1.<i>x</i> &amp; JSF 2.0 Tutorials</a></b></h1>
<div> <!-- Accordion -->
  <h2 class="content_subheading"><a href="http://www.coreservlets.com/JSF-Tutorial/">JSF Tutorial Main Page</a></h2>
  <p>These tutorials have now been divided into two major sections:
  <a href="http://www.coreservlets.com/JSF-Tutorial/jsf1/">the JSF 1.<i>x</i> tutorial</a> and
  <a href="http://www.coreservlets.com/JSF-Tutorial/jsf2/">the JSF 2.0 tutorial</a>.</p>
  <p>
    JSF 1.<i>x</i> topics include JSF intro, page navigation, managed beans,
    expression language, HTML library, validation, Apache MyFaces validators, 
    Apache MyFaces (Tomahawk) custom components, Ajax4jsf,
    JSF data tables, building your own custom components, facelets, and JSTL usage.
  </p>
  <p>
    JSF 2.0 topics include overview of new features, annotations, configuration, Ajax support, 
    explicit page navigation, managed beans, page templating with the integrated facelets support,
    the updated expression language, HTML library, validation, new validators, 
    facelets-based composite components, 
    JSF data tables, and JSTL usage.
  </p>
  <p><font color="blue">Live courses from the author:</font> See the 
  <a href="http://courses.coreservlets.com/public-courses/jsf2/">
    upcoming JSF 2.0 training courses in Maryland</a>, 
    or contact <a href="mailto:hall@coreservlets.com">hall@coreservlets.com</a> for information on a 
    customized version onsite at <i>your</i> organization.
  </p>
</div> <!-- Accordion -->
