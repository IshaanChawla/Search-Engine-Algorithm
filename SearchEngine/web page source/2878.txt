		<h1 class="firstheading"><a href="http://www.codingunit.com/c-tutorial-for-loop-while-loop-break-and-continue" rel="bookmark" title="C Tutorial &#8211; for loop, while loop, break and continue">
		C Tutorial &#8211; for loop, while loop, break and continue</a></h1>

			<div class="entry">
				<div id="watermark">
					<p>In every programming language, thus also in the C programming language, there are circumstances were you want to do the same thing many times. For instance you want to print the same words ten times. You could type ten printf function, but it is easier to use a loop. The only thing you have to do is to setup a loop that execute the same printf function ten times. <span id="more-165"></span><br />
There are three basic types of loops which are:</p>
<ul>
<li>&#8220;for loop&#8221;</li>
<li>&#8220;while loop&#8221;</li>
<li>&#8220;do while loop&#8221;</li>
</ul>
<h2>The for loop</h2>
<p>The &#8220;for loop&#8221; loops from one number to another number and increases by a specified value each time.<br />
The &#8220;for loop&#8221; uses the following structure:</p>
<div id="cl">
<pre><code>
      for (Start value; continue or end condition; increase value)
     		statement;
</code></pre>
</div>
<p>Look at the example below:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;</code>

	int main()
	{
     		int i;
     		for (i = 0; i &lt; 10; i++)
     		{
          		printf ("Hello\n");
         		printf ("World\n");
     		}
     	return 0;
	}</pre>
</div>
<p><strong>Note:</strong> A single instruction can be placed behind the &#8220;for loop&#8221; without the curly brackets.<br />
<strong>Note:</strong> For those who don&#8217;t know printf or need to know more about printf format specifiers, then first a look at our <a href="http://www.codingunit.com/printf-format-specifiers-format-conversions-and-formatted-output" title="printf and format specifiers">printf C language tutorial.</a></p>
<p>Let&#8217;s look at the &#8220;for loop&#8221; from the example: We first start by setting the variable i to 0. This is  where we start to count. Then we say that the for loop must run if the counter i is smaller then ten. Last we say that every cycle i must be increased by one (i++).</p>
<p>In the example we used i++ which is the same as using i = i + 1. This is called incrementing. The instruction i++ adds 1 to i. If you want to subtract 1 from i you can use i<code>--</code>. It is also possible to use ++i or <code>--</code>i. The difference is is that with ++i (prefix incrementing) the one is added before the &#8220;for loop&#8221; tests if i &lt; 10. With i++ (postfix incrementing) the one is added after the test i &lt; 10. In case of a for loop this make no difference, but in while loop test it makes a difference. But before we look at a postfix and prefix increment while loop example, we first look at the while loop.</p>
<h2>The while loop</h2>
<p>The while loop can be used if you don&#8217;t know how many times a loop must run. Here is an example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	int main()

	{
    		int counter, howmuch;

    		scanf("%d", &amp;howmuch);
    		counter = 0;
    		while ( counter &lt; howmuch)
    		{
          		counter++;
          		printf("%d\n", counter);
    		}
    		return 0;
	}

</code></pre>
</div>
<p>Let&#8217;s take a look at the example: First you must always initialize the counter before the while loop starts ( counter = 1). Then the while loop will run if the variable counter is smaller then the variable &#8220;howmuch&#8221;. If the input is ten, then 1 through 10 will be printed on the screen. A last thing you have to remember is to increment the counter inside the loop (counter++). If you forget this the loop becomes infinitive.</p>
<p>As said before (after the for loop example) it makes a difference if prefix incrementing (++i) or postfix incrementing (i++) is used with while loop. Take a look at the following postfix and prefix increment while loop example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	int main(void) {
		int i;
	
		i = 0;
		while(i++ &lt; 5) {
			printf("%d\n", i);
		}
		printf("\n");
		i = 0;
		while(++i &lt; 5) {
			printf("%d\n", i);
		}
		return 0;
	}

</code></pre>
</div>
<p>The output of the postfix and prefix increment example will look like this:</p>
<div id="cl">
<pre><code>
	1
	2
	3
	4
	5

	1
	2
	3
	4

</code></pre>
</div>
<p>i++ will increment the value of i, but is using the pre-incremented value to test against &lt; 5.  That&#8217;s why we get 5 numbers.<br />
++i will increment the value of i, but is using the incremented value to test against &lt; 5. That&#8217;s why we get 4 numbers.</p>
<h2>The do while loop</h2>
<p>The &#8220;do while loop&#8221; is almost the same as the while loop. The &#8220;do while loop&#8221; has the following form:</p>
<div id="cl">
<pre><code>
	do
	{
    		do something;
	}
	while (expression);

</code></pre>
</div>
<p>Do something first and then test if we have to continue. The result is that the loop always runs once. (Because the expression test comes afterward). Take a look at an example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	int main()
	{
     		int counter, howmuch;
     		scanf("%d", &amp;howmuch);
     		counter = 0;
     		do
     		{
          		counter++;
          		printf("%d\n", counter);
     		}
     		while ( counter &lt; howmuch);
     		return 0;
	}

</code></pre>
</div>
<p><strong>Note:</strong> There is a semi-colon behind the while line.</p>
<h2>Break and continue</h2>
<p>To exit a loop you can use the break statement at any time. This can be very useful if you want to stop running a loop because a condition has been met other than the loop end condition. Take a look at the following example:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	int main()
	{
     		int i;

		i = 0;
		while ( i &lt; 20 )
     		{
          		i++;
          		if ( i == 10)
               		break;
     		}
     		return 0;
	}

</code></pre>
</div>
<p>In the example above, the while loop will run, as long i is smaller then twenty. In the while loop there is an if statement that states that if i equals ten the while loop must stop (break).</p>
<p>With &#8220;continue;&#8221; it is possible to skip the rest of the commands in the current loop and start from the top again. (the loop variable must still be incremented). Take a look at the example below:</p>
<div id="cl">
<pre><code>
	#include&lt;stdio.h&gt;

	int main()
	{
		int i;

		i = 0;
     		while ( i &lt; 20 )
     		{
          		i++;
          		continue;
          		printf("Nothing to see\n");
     		}
     		return 0;
	}

</code></pre>
</div>
<p>In the example above, the printf function is never called because of the &#8220;continue;&#8221;.</p>
<p>That was all for now. Don&#8217;t forget to make some example programs of your own, just for practice!</p>

<p>int main()<br />

z = ++v + ++v;</p>

and addition of 2 eleven is 22 so y = 22,</p>
