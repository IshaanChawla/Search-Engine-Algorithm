	<h1>Android Testen und Optimierung von Applikationen</h1>
	
	<strong>Dauer:</strong> 2 Tage<br> 
	<strong>Zielgruppe:</strong> Softwareentwickler und -architekten 
	<hr>
	<p>
		<strong>Schulungsüberblick:</strong><br> 
		Lernen Sie die Analyse und das Testen von Android Applikationen. Diese intensive Schulung führt Sie in Feinheiten von 
		Android Softwaretests ein und zeigt die Möglichkeiten der Android Optimierung anhand von Anwendungsbeispielen ein.
		Erlerntes Wissen wird sofort bei der Entwicklung eines durchgängigen Schulungsbeispiels gefestigt und vertieft. 
		Dabei stehen Ihnen die Referenten mit Rat und Tat zur Seite.
    </p>
    <p>
		<strong>Schulungsunterlagen:</strong><br>
		Sie erhalten umfassende Kursunterlagen.
	</p>
	<p>
		<strong>Schulungsvoraussetzungen:</strong><br>
    	Vorausgesetzt werden gute Kenntnisse der Java-Programmierung sowie Grundwissen der Androidprogrammierung.
   	 	Bei Anmeldung zur Schulung erhalten Sie Vorbereitungsmaterial, welches Sie gegebenenfalls zur Auffrischung Ihrer 
   	 	Java Kenntnisse nutzen können.
    </p>

	<h3>Agenda</h3>
	<ul>
		<li><span class="emphasis"><em>Testing as methodology</em></span>
			<ul>
				<li>Unit tests</li>
				<li>Functional and integration tests</li>
				<li>Test driven development</li>
				<li>Continuous integration</li>
			</ul>
		</li>
		<li><span class="emphasis"><em>Unit Testing with JUnit </em></span>
			<ul>
				<li>JUnit 4.x and JUnit 3.x </li>
				<li>Testrunner </li>
				<li>Android test projects</li>
				<li>Mock Frameworks for JUnit testing</li>
			</ul>
		</li>
		<li><span class="emphasis"><em>Running tests in Eclipse</em></span>
			<ul>
				<li>Settings for bug detection</li>
				<li>Efficient testing in Eclipse  </li>
			</ul>
		</li>
		<li><span class="emphasis"><em>Android testing framework</em></span>
			<ul>
				<li>InstrumentationTestRunner</li>
				<li>Mock object classes</li>
				<li>Activity testing</li>
				<li>Service testing</li>
				<li>ContentProvider testing</li>
			</ul>
		</li>
		<li><span class="emphasis"><em>Android user interface and stress testing</em></span>
			<ul>
				<li>User interface testing with uiautomator</li>
				<li>Stress testing with Monkey</li>
				<li>Scripting test cases with Monkeyrunner</li>
			</ul>
		</li>

		<li><span class="emphasis"><em>Android tools for application optimization</em></span>
			<ul>
				<li>StrictMode</li>
				<li>Using on-device Developer Options</li>
				<li>Lint</li>
				<li>TraceView</li>
				<li>HierarchyViewer</li>
				<li>Systrace</li>
			</ul>
		</li>

		<li><span class="emphasis"><em>Android tools for layout optimization</em></span>
			<ul>
				<li>Pixel Perfect perspective</li>

				<li>Profile GPU rendering</li>
				<li>Analyzing Overdraw</li>
			</ul>
		</li>

		<li><span class="emphasis"><em>Analyzing memory allocation with heap dumps</em></span>
			<ul>
				<li>Creating heap dumps</li>
				<li>Using Eclipse MAT for the analyzes</li>
			</ul>
		</li>

		<li><span class="emphasis"><em>Open Source testing frameworks</em></span>
			<ul>

				<li>RoboGuice for dependency injection</li>
			</ul>

		<li><span class="emphasis"><em>Build and test automation and continuous integration </em></span>
