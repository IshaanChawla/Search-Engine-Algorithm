<p>Write a full program or a function that takes a positive integer <code>N</code> as input via STDIN/command line/ARGV or function arguments and prints an ASCII double knot corresponding to the <code>N</code> to STDOUT.</p>

<p>The ASCII double knot looks like this:</p>

<pre><code>  __  __  __  __  __  __
 /  \/  \/  \/  \/  \/  \
| /\/ /\/ /\/ /\/ /\/ /\ |
| \/ /\/ /\/ /\/ /\/ /\/ |
 \ \/\ \/\ \/\ \/\ \/\ \/
 /\ \/\ \/\ \/\ \/\ \/\ \
| /\/ /\/ /\/ /\/ /\/ /\ |
| \/ /\/ /\/ /\/ /\/ /\/ |
 \__/\__/\__/\__/\__/\__/
</code></pre>

<p>The above is for <code>N = 6</code></p>

<p>Here are a few more double knots for other values of <code>N</code>:</p>

<p>If <code>N = 1</code>, the output double knot looks like:</p>

<pre><code>  __
 /  \
| /\ |
| \/ |
 \ \/
 /\ \
| /\ |
| \/ |
 \__/
</code></pre>

<p>For <code>N = 2</code>, its</p>

<pre><code>  __  __ 
 /  \/  \
| /\/ /\ |
| \/ /\/ |
 \ \/\ \/
 /\ \/\ \
| /\/ /\ |
| \/ /\/ |
 \__/\__/
</code></pre>

<p>For <code>N = 3</code>, its</p>

<pre><code>  __  __  __ 
 /  \/  \/  \
| /\/ /\/ /\ |
| \/ /\/ /\/ |
 \ \/\ \/\ \/
 /\ \/\ \/\ \
| /\/ /\/ /\ |
| \/ /\/ /\/ /
 \__/\__/\__/
</code></pre>

<p>and similarly, the pattern continues and any larger value of <code>N</code>.</p>

<p><strong>Details</strong>:</p>

<ul>
<li>The input is always a positive integer greater than 0.</li>
<li>Trailing newline is optional</li>
<li>There should either be no trailing spaces in each line, or enough trailing spaces such that each row's length is <code>4*N + 2</code>.</li>
<li>There should never be any leading space that aren't part of the specified pattern.</li>
</ul>

<p>This is <a href="/questions/tagged/code-golf" class="post-tag" title="show questions tagged &#39;code-golf&#39;" rel="tag">code-golf</a>, so shortest code in bytes wins.</p>

<hr>

<p>I am converting this into a series of ASCII art challenges and thus adding a leader board for the series (snippet by Martin). To make sure that your answers show up, please start every answer with a headline, using the following Markdown template:</p>

<pre><code># Language Name, N bytes
</code></pre>

<p>where N is the size of your submission. If you improve your score, you can keep old scores in the headline, by striking them through. For instance:</p>

<pre><code># Ruby, &lt;s&gt;104&lt;/s&gt; &lt;s&gt;101&lt;/s&gt; 96 bytes
</code></pre>

var SIZE_REG = /\d+(?=[^\d&amp;]*(?:&amp;lt;(?:s&amp;gt;((?!&amp;gt;).)*&amp;lt;\/s&amp;gt;|((?!&amp;gt;).)+&amp;gt;)[^\d&amp;]*)*$)/;

function shouldHaveHeading(a) {

    pass &amp;= LANGUAGE_REG.test(a.body_markdown);
  } catch (ex) {}
  return pass;
}

function shouldHaveScore(a) {
  var pass = false;

  return pass;
}

function getAuthorName(a) {

function getAuthorId(a) {

function process() {

    return aB - bB
  });

  var users = {};

<h1>CJam, <s>103</s> <s>92</s> <s>83</s> <s>69</s> <s>66</s> 57 bytes</h1>

<pre><code>"O5àäªÞáKZò,_÷%SJ«Ò"248b5b"/ \_|"f=9/4/~ri*1&gt;W&lt;+2m&lt;zN*
</code></pre>

<p>Try it online in the <a href="http://cjam.aditsu.net/#code=%22O5%C3%A0%C3%A4%C2%99%C2%AA%C2%96%C3%9E%C3%A1KZ%C3%B2%2C_%C3%B7%25SJ%C2%AB%C3%92%C2%8E%22248b5b%22%2F%20%5C_%7C%22f%3D9%2F4%2F~ri*1%3EW%3C%2B2m%3CzN*&amp;input=12" rel="nofollow">CJam interpreter</a>.</p>

<h3>Idea</h3>

<p>The pattern becomes a lot more obvious once we transpose rows with columns (padded to the right with spaces).</p>

<p>For input <strong>3</strong>, for example, this gives:</p>

<pre><code>  ||  || 
 /  \/  \
_ /\ \/\_
_ \/\ \/_
 \/ /\/ /
 / /\/ /\
_ /\ \/\_
_ \/\ \/_
 \/ /\/ /
 / /\/ /\
_ /\ \/\_
_ \/\ \/_
 \  /\  /
  ||  || 
</code></pre>

<p>The first two and last two rows are special, but the others repeat the same pattern over and over again.</p>

<p>Thus, for input <strong>N</strong>, all we have to do is repeat</p>

<pre><code> / /\/ /\
_ /\ \/\_
_ \/\ \/_
 \/ /\/ /
</code></pre>

<p><strong>N</strong> times, replace the first row with</p>

<pre><code>  ||  || 
 /  \/  \
</code></pre>

<p>the last row with</p>

<pre><code> \  /\  /
  ||  || 
</code></pre>

<p>and, finally, transpose rows with columns.</p>

<h3>Code</h3>

<pre><code>"O5àäªÞáKZò,_÷%SJ«Ò"

248b  e# Convert the string (array of code points in base 248) to an integer.
5b    e# Convert the integer to the array of its digits in base 5.

      e# This pushes the following array:

      e# [ 1 2 1 1 0 2 1 1 0 1 1 4 4 1 1 4 4 1 1 1 4 4 1 1
      e#   4 4 1 1 0 1 1 2 0 1 1 2 1 0 1 0 2 0 1 0 2 3 1 0
      e#   2 1 2 0 2 3 3 1 2 0 2 1 2 0 3 1 2 0 1 0 2 0 1 0 ]

"/ \_"

f=    e# Retrieve the chars that correspond to the indexes in the array.

      e# This pushes the following string:

      e# " \  /\  /  ||  ||   ||  ||  /  \/  \ / /\/ /\_ /\ \/\__ \/\ \/_ \/ /\/ /"

9/4/~ e# Divide into groups of 4 strings of length 9.

      e# This pushes the following arrays:

      e# [ " \  /\  /"   "  ||  || "   "  ||  || "   " /  \/  \" ]
      e# [ " / /\/ /\"   "_ /\ \/\_"   "_ \/\ \/_"   " \/ /\/ /" ]

ri    e# Read an integer from STDIN.
*     e# Repeat the last array that many times.
1&gt;W&lt;  e# Remove the first and last string from the array.
+     e# Concatenate both arrays.
2m&lt;   e# Rotate the concatenated arrays two elements to the left.
z     e# Zip the array of strings (transpose rows with columns).
N*    e# Join the strings of the array, separating by linefeeds.
</code></pre>

<h1>JavaScript (<em>ES6</em>), 165 <s>166</s></h1>

<p>Knot elements:</p>

<ul>
<li>2 top rows, based on '  __' and '/  \'  repeated</li>
<li>2 outer rows, based on '/\/ ' repeated and enclosed in '| ... |'</li>
<li>2 inner rows, based on '\/\ ' repeated and enclosed in '.../' and '/...\'</li>
<li>the same 2 outer rows above</li>
<li>1 bottom row based on '\__/' repeated</li>
</ul>

<p><strong>Ungolfed</strong></p>

<pre class="lang-js prettyprint-override"><code>K=r=&gt;
{
  R = x =&gt; x.repeat(r);
  S = d =&gt; R(elem).slice(d,d-2); // use global 'elem' that must be assigned before the call
  elem = '/\\/ '
  outer = '\n| ' + S(0) + ' |\n| ' + S(1) + ' |\n '
  elem = '\\ \\/'
  inner = '\\' + S(1) + '/\n /' + S(0) + '\\'
  return R('  __') + '\n ' + R('/  \\') + outer
  + inner + outer + R('\\__/')
}
</code></pre>

<p><strong>Golfed</strong> </p>

<p><div class="snippet" data-lang="js" data-hide="false">
<div class="snippet-code">
<pre class="snippet-code-js lang-js prettyprint-override"><code>K=r=&gt;(S=d=&gt;(R=e=&gt;e.repeat(r))(e).slice(d,d-2),o=`
| ${S(0,e='/\\/ ')} |
| ${S(1)} |
 `,R('  __')+`
 ${R('/  \\')+o}\\${S(1,e='\\ \\/')}/
 /${S(0)}\\${o}`+R('\\__/'))

// TEST

function go() {
  O.innerHTML=K(I.value)
}

go()</code></pre>
<pre class="snippet-code-html lang-html prettyprint-override"><code>&lt;input id=I value=5&gt;&lt;button onclick='go()'&gt;-&gt;&lt;/button&gt;&lt;br&gt;
&lt;pre id=O&gt;&lt;/pre&gt;</code></pre>
</div>
</div>
</p>

<h1>Python, 156</h1>

<p>This was good fun to program, my first code golf!</p>

<pre><code>def k(n):
 a=n-1
 b='/\\/ '*a
 c=r'\/\ '*a
 d='\n| '+b+'/\\ |\n| \\/ '+b+'|\n'
 print'  __'*n+'\n '+'/  \\'*n+d+' \\ '+c+'\\/\n /\\ '+c+'\\'+d+' '+'\\__/'*n
</code></pre>

<p>save it in an external text editor as n.py in order to remove last newline character in order to get from 157 to 156 bytes, open and run the module, and call k(n).</p>

<h2>PowerShell, <strike>228</strike> <strike>207</strike> 181 bytes</h2>

<pre><code>%{$a=$_;filter x($s){(1..$a|%{$s})-join''}"  "+(x "__  ");" "+(x "/  \");$b="|"+(x " /\/")+"`b |";$b;$c="| "+(x "\/ /")+"`b|";$c;" "+(x "\ \/");" "+(x "/\ \");$b;$c;" "+(x "\__/");}
</code></pre>

<p>Reminder - PowerShell doesn't wait on stdin automatically. You have to pipe something in, e.g. <code>3|%{...}</code></p>
