&#x2022; Find any variable or parameter that expects

public static double integrate(Integrable function,
double x1, double x2,
int numSlices){
if (numSlices &lt; 1) {
numSlices = 1;
}
double delta = (x2 - x1)/numSlices;
double start = x1 + delta/2;
double sum = 0;
for(int i=0; i&lt;numSlices; i++) {
sum += delta * function.eval(start + delta * i);
}
return(sum);
}

    General Lambda Principle
&#x2022; Interfaces in Java 8 are same as in Java 7
&#x2013; Integrable was the same as it would be in Java 7, except that
you can (should) optionally use @FunctionalInterface
&#x2022; To catch errors (multiple methods) at compile time
&#x2022; To express design intent (developers should use lambdas)
&#x2022; Code that uses interfaces is the same in Java 8
as in Java 7
&#x2013; I.e., the definition of integrate is exactly the same as you
would have written it in Java 7. The author of integrate must
know that the real method name is eval.
&#x2022; Code that calls methods that expect 1-method
interfaces can now use lambdas
&#x2013; MathUtilities.integrate(x -&gt; Math.sin(x), 0, Math.PI, &#x2026;);
